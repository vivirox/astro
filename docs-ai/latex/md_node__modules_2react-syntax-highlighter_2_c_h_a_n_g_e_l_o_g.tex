When paired with {\ttfamily show\+Line\+Numbers=\{true\}}, {\ttfamily show\+Inline\+Line\+Numbers=\{true\}} injects line numbers into each wrapped line of code, instead of creating a separate {\ttfamily \texorpdfstring{$<$}{<}code\texorpdfstring{$>$}{>}\texorpdfstring{$<$}{<}/code\texorpdfstring{$>$}{>}} block for the numbers. This allows line numbering to work with virtualized renderers such as {\ttfamily react-\/syntax-\/highlighter-\/virtualized-\/renderer}.
