Non-\/interactive HTML elements indicate {\itshape content} and {\itshape containers} in the user interface. Non-\/interactive elements include {\ttfamily \texorpdfstring{$<$}{<}main\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}area\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}h1\texorpdfstring{$>$}{>}} (,{\ttfamily \texorpdfstring{$<$}{<}h2\texorpdfstring{$>$}{>}}, etc), {\ttfamily \texorpdfstring{$<$}{<}img\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}li\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}ul\texorpdfstring{$>$}{>}} and {\ttfamily \texorpdfstring{$<$}{<}ol\texorpdfstring{$>$}{>}}.

Interactive HTML elements indicate {\itshape controls} in the user interface. Interactive elements include {\ttfamily \texorpdfstring{$<$}{<}a href\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}button\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}input\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}select\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}textarea\texorpdfstring{$>$}{>}}.

\href{https://www.w3.org/TR/wai-aria-1.1/\#usage_intro}{\texttt{ WAI-\/\+ARIA roles}} should not be used to convert a non-\/interactive element to an interactive element.
