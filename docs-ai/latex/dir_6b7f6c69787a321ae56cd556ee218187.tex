\doxysubsection{Detailed Description}
A default plugin of Expressive Code, an engine for presenting source code on the web.

It performs syntax highlighting of your code blocks using Shiki, which uses the same engine as VS Code to provide accurate syntax highlighting for over 100 languages.\hypertarget{README.md_autotoc_md3313}{}\doxysubsection{\texorpdfstring{Documentation}{Documentation}}\label{README.md_autotoc_md3313}
\href{https://expressive-code.com/key-features/syntax-highlighting/}{\texttt{ Read this plugin\textquotesingle{}s documentation}} on the Expressive Code website to learn more about its features.\hypertarget{README.md_autotoc_md3314}{}\doxysubsection{\texorpdfstring{Installation (not required)}{Installation (not required)}}\label{README.md_autotoc_md3314}
No installation is required. This package is {\bfseries{installed by default}} by our higher-\/level packages.
