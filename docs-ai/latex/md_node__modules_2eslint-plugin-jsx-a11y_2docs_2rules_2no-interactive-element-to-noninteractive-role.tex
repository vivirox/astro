Interactive HTML elements indicate {\itshape controls} in the user interface. Interactive elements include {\ttfamily \texorpdfstring{$<$}{<}a href\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}button\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}input\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}select\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}textarea\texorpdfstring{$>$}{>}}.

Non-\/interactive HTML elements and non-\/interactive ARIA roles indicate {\itshape content} and {\itshape containers} in the user interface. Non-\/interactive elements include {\ttfamily \texorpdfstring{$<$}{<}main\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}area\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}h1\texorpdfstring{$>$}{>}} (,{\ttfamily \texorpdfstring{$<$}{<}h2\texorpdfstring{$>$}{>}}, etc), {\ttfamily \texorpdfstring{$<$}{<}img\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}li\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}ul\texorpdfstring{$>$}{>}} and {\ttfamily \texorpdfstring{$<$}{<}ol\texorpdfstring{$>$}{>}}.

\href{https://www.w3.org/TR/wai-aria-1.1/\#usage_intro}{\texttt{ WAI-\/\+ARIA roles}} should not be used to convert an interactive element to a non-\/interactive element. \hypertarget{md_node__modules_2eslint-plugin-jsx-a11y_2docs_2rules_2no-interactive-element-to-noninteractive-role_autotoc_md13901}{}\doxysection{\texorpdfstring{Rule options}{Rule options}}\label{md_node__modules_2eslint-plugin-jsx-a11y_2docs_2rules_2no-interactive-element-to-noninteractive-role_autotoc_md13901}
The recommended options for this rule allow the {\ttfamily tr} element to be given a role of {\ttfamily presentation} (or its semantic equivalent {\ttfamily none}). Under normal circumstances, an element with an interactive role should not be semantically neutralized with {\ttfamily presentation} (or {\ttfamily none}).

Options are provided as an object keyed by HTML element name; the value is an array of interactive roles that are allowed on the specified element.
