\doxysubsection{Detailed Description}
{\ttfamily @upstash/redis} is an HTTP/\+REST based Redis client for typescript, built on top of \href{https://docs.upstash.com/features/restapi}{\texttt{ Upstash REST API}}.

\href{https://github.com/upstash/upstash-redis/actions/workflows/tests.yaml}{\texttt{ }}  

\begin{DoxyNote}{Note}
{\bfseries{This project is in GA Stage.}} ~\newline
~\newline
 The Upstash Professional Support fully covers this project. It receives regular updates, and bug fixes. The Upstash team is committed to maintaining and improving its functionality.
\end{DoxyNote}
It is the only connectionless (HTTP based) Redis client and designed for\+:


\begin{DoxyItemize}
\item Serverless functions (AWS Lambda ...)
\item Cloudflare Workers (see \href{https://github.com/upstash/upstash-redis/tree/main/examples/cloudflare-workers}{\texttt{ the example}})
\item Fastly Compute@\+Edge (see \href{https://github.com/upstash/upstash-redis/tree/main/examples/fastly}{\texttt{ the example}})
\item Next.\+js, Jamstack ...
\item Client side web/mobile applications
\item Web\+Assembly
\item and other environments where HTTP is preferred over TCP.
\end{DoxyItemize}

\begin{DoxyItemize}

\end{DoxyItemize}
