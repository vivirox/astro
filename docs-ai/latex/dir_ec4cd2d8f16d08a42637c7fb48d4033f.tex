This parser allows us to lint the {\ttfamily \texorpdfstring{$<$}{<}template\texorpdfstring{$>$}{>}} of {\ttfamily .vue} files. We can make mistakes easily on {\ttfamily \texorpdfstring{$<$}{<}template\texorpdfstring{$>$}{>}} if we use complex directives and expressions in the template.
