\hypertarget{README.md_autotoc_md30971}{}\doxysubsubsection{\texorpdfstring{{\ttfamily use\+SWRStore(store, args, options)}}{{\ttfamily use\+SWRStore(store, args, options)}}}\label{README.md_autotoc_md30971}
Subscribes to an SWR store, passing {\ttfamily args}, which are received by the corresponding store for data-\/fetching and cache updates.

{\ttfamily options} has the following properties\+:


\begin{DoxyItemize}
\item {\ttfamily initial\+Data}\+: Allows lazy hydration when reading the store. If the store does not have cache, {\ttfamily initial\+Data} hydrates the cache and attempts a revalidation. If no {\ttfamily initial\+Data} is provided, defaults to store\textquotesingle{}s {\ttfamily options.\+initial\+Data}.
\item {\ttfamily should\+Revalidate}\+: If {\ttfamily true}, goes through the revalidation process when reading through the cache. Defaults to {\ttfamily true}.
