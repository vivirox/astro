This module accepts a stream instead of being one and returns a promise instead of using a callback. The API is simpler and it only supports returning a string, {\ttfamily Buffer}, an {\ttfamily Array\+Buffer} or an array. It doesn\textquotesingle{}t have a fragile type inference.
