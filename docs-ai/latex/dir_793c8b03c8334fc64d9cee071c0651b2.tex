\doxysubsection{Detailed Description}
\begin{quote}
Delete files and directories using \href{https://github.com/sindresorhus/globby\#globbing-patterns}{\texttt{ globs}} \end{quote}


Similar to \href{https://github.com/isaacs/rimraf}{\texttt{ rimraf}}, but with a Promise API and support for multiple files and globbing. It also protects you against deleting the current working directory and above.\hypertarget{readme.md_autotoc_md11742}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md11742}

\hypertarget{readme.md_autotoc_md11744}{}\doxysubsection{\texorpdfstring{Beware}{Beware}}\label{readme.md_autotoc_md11744}
The glob pattern {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}} matches all children and {\itshape the parent}.

So this won\textquotesingle{}t work\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{deleteSync(['public/assets/**',\ '!public/assets/goat.png']);}

\end{DoxyCode}


You have to explicitly ignore the parent directories too\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{deleteSync(['public/assets/**',\ '!public/assets',\ '!public/assets/goat.png']);}

\end{DoxyCode}


To delete all subdirectories inside {\ttfamily public/}, you can do\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{deleteSync(['public/*/']);}

\end{DoxyCode}

\hypertarget{readme.md_autotoc_md11752}{}\doxysubsubsubsubsection{\texorpdfstring{dot}{dot}}\label{readme.md_autotoc_md11752}
Type\+: {\ttfamily boolean}\textbackslash{} Default\+: {\ttfamily false}

Allow patterns to match files/folders that start with a period ({\ttfamily .}).

This option is passed through to \href{https://github.com/mrmlnc/fast-glob\#dot}{\texttt{ {\ttfamily fast-\/glob}}}.

Note that an explicit dot in a portion of the pattern will always match dot files.
