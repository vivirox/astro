This package contains two extensions that add support for GFM footnote syntax in markdown to \href{https://github.com/syntax-tree/mdast}{\texttt{ mdast}}. These extensions plug into \href{https://github.com/syntax-tree/mdast-util-from-markdown}{\texttt{ {\ttfamily mdast-\/util-\/from-\/markdown}}} (to support parsing footnotes in markdown into a syntax tree) and \href{https://github.com/syntax-tree/mdast-util-to-markdown}{\texttt{ {\ttfamily mdast-\/util-\/to-\/markdown}}} (to support serializing footnotes in syntax trees to markdown).

GFM footnotes were \href{https://github.blog/changelog/2021-09-30-footnotes-now-supported-in-markdown-fields/}{\texttt{ announced September 30, 2021}} but are not specified. Their implementation on github.\+com is currently buggy. The bugs have been reported on \href{https://github.com/github/cmark-gfm}{\texttt{ {\ttfamily cmark-\/gfm}}}.\hypertarget{readme.md_autotoc_md22400}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md22400}
You can use these extensions when you are working with {\ttfamily mdast-\/util-\/from-\/markdown} and {\ttfamily mdast-\/util-\/to-\/markdown} already.

When working with {\ttfamily mdast-\/util-\/from-\/markdown}, you must combine this package with \href{https://github.com/micromark/micromark-extension-gfm-footnote}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/footnote}}}.

When you don’t need a syntax tree, you can use \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}} directly with {\ttfamily micromark-\/extension-\/gfm-\/footnote}.

When you are working with syntax trees and want all of GFM, use \href{https://github.com/syntax-tree/mdast-util-gfm}{\texttt{ {\ttfamily mdast-\/util-\/gfm}}} instead.

All these packages are used \href{https://github.com/remarkjs/remark-gfm}{\texttt{ {\ttfamily remark-\/gfm}}}, which focusses on making it easier to transform content by abstracting these internals away.

This utility does not handle how markdown is turned to HTML. That’s done by \href{https://github.com/syntax-tree/mdast-util-to-hast}{\texttt{ {\ttfamily mdast-\/util-\/to-\/hast}}}. If your content is not in English, you should configure that utility.\hypertarget{readme.md_autotoc_md22401}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22401}
