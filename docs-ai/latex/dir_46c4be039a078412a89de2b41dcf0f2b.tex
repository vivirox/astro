{\ttfamily Separator} objects can be used in the {\ttfamily choices} array to render non-\/selectable lines in the choice list. By default it\textquotesingle{}ll render a line, but you can provide the text as argument (`new Separator('-- Dependencies --\textquotesingle{})\`{}). This option is often used to add labels to groups within long list of options.\hypertarget{README.md_autotoc_md3821}{}\doxysubsubsubsection{\texorpdfstring{{\ttfamily Choice} object}{{\ttfamily Choice} object}}\label{README.md_autotoc_md3821}

Here\textquotesingle{}s each property\+:


\begin{DoxyItemize}
\item {\ttfamily value}\+: The value is what will be returned by {\ttfamily await select()}.
\item {\ttfamily name}\+: This is the string displayed in the choice list.
\item {\ttfamily description}\+: Option for a longer description string that\textquotesingle{}ll appear under the list when the cursor highlight a given choice.
\item {\ttfamily short}\+: Once the prompt is done (press enter), we\textquotesingle{}ll use {\ttfamily short} if defined to render next to the question. By default we\textquotesingle{}ll use {\ttfamily name}.
\item {\ttfamily disabled}\+: Disallow the option from being selected. If {\ttfamily disabled} is a string, it\textquotesingle{}ll be used as a help tip explaining why the choice isn\textquotesingle{}t available.
\end{DoxyItemize}

{\ttfamily choices} can also be an array of string, in which case the string will be used both as the {\ttfamily value} and the {\ttfamily name}.\hypertarget{README.md_autotoc_md3822}{}\doxysubsubsection{\texorpdfstring{Theming}{Theming}}\label{README.md_autotoc_md3822}
