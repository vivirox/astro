\doxysubsection{Detailed Description}

\hypertarget{README.md_autotoc_md18941}{}\doxysubsubsection{\texorpdfstring{{\ttfamily isexe(path, \mbox{[}options\mbox{]}, \mbox{[}callback\mbox{]})}}{{\ttfamily isexe(path, \mbox{[}options\mbox{]}, \mbox{[}callback\mbox{]})}}}\label{README.md_autotoc_md18941}
Check if the path is executable. If no callback provided, and a global {\ttfamily Promise} object is available, then a Promise will be returned.

Will raise whatever errors may be raised by {\ttfamily fs.\+stat}, unless {\ttfamily options.\+ignore\+Errors} is set to true.\hypertarget{README.md_autotoc_md18942}{}\doxysubsubsection{\texorpdfstring{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}}\label{README.md_autotoc_md18942}
Same as {\ttfamily isexe} but returns the value and throws any errors raised.\hypertarget{README.md_autotoc_md18943}{}\doxysubsubsection{\texorpdfstring{Options}{Options}}\label{README.md_autotoc_md18943}

\begin{DoxyItemize}
\item {\ttfamily ignore\+Errors} Treat all errors as "{}no, this is not executable"{}, but don\textquotesingle{}t raise them.
\item {\ttfamily uid} Number to use as the user id
\item {\ttfamily gid} Number to use as the group id
\item {\ttfamily path\+Ext} List of path extensions to use instead of {\ttfamily PATHEXT} environment variable on Windows.
