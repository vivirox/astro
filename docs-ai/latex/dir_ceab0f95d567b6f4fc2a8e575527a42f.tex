Benchmark your code easily with Tinybench, a simple, tiny and light-\/weight {\ttfamily 7KB} ({\ttfamily 2KB} minified and gzipped) benchmarking library! You can run your benchmarks in multiple Java\+Script runtimes, Tinybench is completely based on the Web APIs with proper timing using {\ttfamily process.\+hrtime} or {\ttfamily performance.\+now}.
