\doxysubsection{Detailed Description}
\hypertarget{README.md_autotoc_md18945}{}\doxysubsection{\texorpdfstring{What is it}{What is it}}\label{README.md_autotoc_md18945}
A simple package for validating strings denoting dates and time, including ISO 8601 format. The package provides the following functions\+:


\begin{DoxyEnumerate}
\item {\bfseries{Date validation}}. YYYY-\/\+MM-\/\+DD format from 0001-\/01-\/01 to 9999-\/12-\/31, leap year friendly. Custom digit separators and null separators supported\+: YYYY/\+MM/\+DD or YYYYMMDD is no problem.
\item {\bfseries{Time validation}}. HH\+:mm\+:ss.\+fff±hh\+:mm format, seconds, fractions of seconds and timezone offset being optional. Custom digit separators supported for HHmmss as well (no custom separator for fractions, it is dot).
\end{DoxyEnumerate}

{\bfseries{Caveat}}\+: do not use \textquotesingle{}-\/\textquotesingle{} and \textquotesingle{}+\textquotesingle{} as separators when validating time with timezone. I am reluctant to fix this unless it is an issue.


\begin{DoxyCode}{0}
\DoxyCodeLine{isValidTime('14-\/45-\/15.000+00-\/00',\ '-\/',\ true);}
\DoxyCodeLine{//\ will\ yield\ wrong\ result}

\end{DoxyCode}



\begin{DoxyEnumerate}
\item {\bfseries{Year-\/month validation}}.
\item {\bfseries{ISO 8601 datestring validation}} with timezones, with and without separators\+:
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item 2019-\/07-\/09T15\+:03\+:36.\+000+00\+:00

\hypertarget{README.md_autotoc_md18950}{}\doxysubsubsection{\texorpdfstring{Time validation}{Time validation}}\label{README.md_autotoc_md18950}
Time string in HH\+:mm\+:ss.\+fff±hh\+:mm format can be validated with the {\bfseries{is\+Valid\+Time}} function. Seconds and fractions are optional. However, if using fractions min number of numbers is 1 and max is 9. Zone offset is optional as well, its check is switched off by default.

\DoxyCodeLine{isValidTime('13:00');}
\DoxyCodeLine{//\ true}
\DoxyCodeLine{}
\DoxyCodeLine{isValidTime('13:00:00');}
\DoxyCodeLine{//\ true}
\DoxyCodeLine{}
\DoxyCodeLine{isValidTime('13:00:00.000000000');}
\DoxyCodeLine{//\ true}
\DoxyCodeLine{}
