This package is a utility that lets you find patterns ({\ttfamily string}, {\ttfamily Reg\+Exp}) in text and replace them with nodes.\hypertarget{readme.md_autotoc_md22304}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md22304}
This utility is typically useful when you have regexes and want to modify mdast. One example is when you have some form of “mentions” (such as {\ttfamily /@(\mbox{[}a-\/z\mbox{]}\mbox{[}\+\_\+a-\/z0-\/9\mbox{]})\textbackslash{}b/gi}) and want to create links to persons from them.

A similar package, \href{https://github.com/syntax-tree/hast-util-find-and-replace}{\texttt{ {\ttfamily hast-\/util-\/find-\/and-\/replace}}} does the same but on \href{https://github.com/syntax-tree/hast}{\texttt{ hast}}.\hypertarget{readme.md_autotoc_md22305}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22305}
