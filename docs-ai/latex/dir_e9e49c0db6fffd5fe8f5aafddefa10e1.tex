\doxysubsection{Detailed Description}
This package provides a feed loader for Astro. It allows you to load and parse RSS, RDF Atom feeds, and use the data in your Astro site.\hypertarget{README.md_autotoc_md181}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md181}
