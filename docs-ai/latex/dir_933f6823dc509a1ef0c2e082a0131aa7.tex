\doxysubsection{Detailed Description}
This is a port of Joergen Ibsen\textquotesingle{}s \href{https://bitbucket.org/jibsen/tinf}{\texttt{ tiny inflate}} to Java\+Script. Minified it is about 3KB, or 1.\+3\+KB gzipped. While being very small, it is also reasonably fast (about 30\% -\/ 50\% slower than \href{https://github.com/nodeca/pako}{\texttt{ pako}} on average), and should be good enough for many applications. If you need the absolute best performance, however, you\textquotesingle{}ll need to use a larger library such as pako that contains additional optimizations.\hypertarget{readme.md_autotoc_md32816}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{readme.md_autotoc_md32816}
