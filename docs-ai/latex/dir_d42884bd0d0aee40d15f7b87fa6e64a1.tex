\doxysubsection{Detailed Description}
Slim progress bars for Ajax\textquotesingle{}y applications. Inspired by Google, You\+Tube, and Medium.\hypertarget{Readme.md_autotoc_md24724}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{Readme.md_autotoc_md24724}
Add \href{http://ricostacruz.com/nprogress/nprogress.js}{\texttt{ nprogress.\+js}} and \href{http://ricostacruz.com/nprogress/nprogress.css}{\texttt{ nprogress.\+css}} to your project.


\begin{DoxyCode}{0}
\DoxyCodeLine{<script\ src='nprogress.js'></script>}
\DoxyCodeLine{<link\ rel='stylesheet'\ href='nprogress.css'/>}

\end{DoxyCode}


NProgress is available via \href{http://bower.io/search/?q=nprogress}{\texttt{ bower}} and \href{https://www.npmjs.org/package/nprogress}{\texttt{ npm}} and \href{http://spmjs.io/package/nprogress}{\texttt{ spm}}. \begin{DoxyVerb}$ bower install --save nprogress
$ npm install --save nprogress
\end{DoxyVerb}
\hypertarget{Readme.md_autotoc_md24725}{}\doxysubsection{\texorpdfstring{Basic usage}{Basic usage}}\label{Readme.md_autotoc_md24725}
Simply call {\ttfamily start()} and {\ttfamily done()} to control the progress bar.


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{NProgress.start();}
\DoxyCodeLine{NProgress.done();}

\end{DoxyCode}


Using \href{https://github.com/rails/turbolinks}{\texttt{ Turbolinks}} or similar? Ensure you\textquotesingle{}re using Turbolinks 1.\+3.\+0+, and use this\+: (explained \href{https://github.com/rstacruz/nprogress/issues/8\#issuecomment-23010560}{\texttt{ here}})


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{\$(document).on('page:fetch',\ \ \ function()\ \{\ NProgress.start();\ \});}
\DoxyCodeLine{\$(document).on('page:change',\ \ function()\ \{\ NProgress.done();\ \});}
\DoxyCodeLine{\$(document).on('page:restore',\ function()\ \{\ NProgress.remove();\ \});}

\end{DoxyCode}
\hypertarget{Readme.md_autotoc_md24726}{}\doxysubsection{\texorpdfstring{Ideas}{Ideas}}\label{Readme.md_autotoc_md24726}

\begin{DoxyItemize}
\item Add progress to your Ajax calls! Bind it to the j\+Query {\ttfamily ajax\+Start} and {\ttfamily ajax\+Stop} events.
\item Make a fancy loading bar even without Turbolinks/\+Pjax! Bind it to {\ttfamily \$(document).ready} and {\ttfamily \$(window).load}.
\end{DoxyItemize}\hypertarget{Readme.md_autotoc_md24727}{}\doxysubsection{\texorpdfstring{Advanced usage}{Advanced usage}}\label{Readme.md_autotoc_md24727}
{\bfseries{Percentages\+:}} To set a progress percentage, call {\ttfamily .set(n)}, where {\itshape n} is a number between {\ttfamily 0..1}.


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{NProgress.set(0.0);\ \ \ \ \ //\ Sorta\ same\ as\ .start()}
\DoxyCodeLine{NProgress.set(0.4);}
\DoxyCodeLine{NProgress.set(1.0);\ \ \ \ \ //\ Sorta\ same\ as\ .done()}

\end{DoxyCode}


{\bfseries{Incrementing\+:}} To increment the progress bar, just use {\ttfamily .inc()}. This increments it with a random amount. This will never get to 100\%\+: use it for every image load (or similar).


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{NProgress.inc();}

\end{DoxyCode}


If you want to increment by a specific value, you can pass that as a parameter\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{NProgress.inc(0.2);\ \ \ \ //\ This\ will\ get\ the\ current\ status\ value\ and\ adds\ 0.2\ until\ status\ is\ 0.994}

\end{DoxyCode}


{\bfseries{Force-\/done\+:}} By passing {\ttfamily true} to {\ttfamily done()}, it will show the progress bar even if it\textquotesingle{}s not being shown. (The default behavior is that {\itshape .done()} will not do anything if {\itshape .start()} isn\textquotesingle{}t called)


\begin{DoxyCode}{0}
\DoxyCodeLine{\ js}
\DoxyCodeLine{NProgress.done(true);}

\end{DoxyCode}
