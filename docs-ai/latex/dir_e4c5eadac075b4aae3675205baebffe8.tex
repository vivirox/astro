\doxysubsection{Detailed Description}

Compatibility\+: Node.\+js version 3+ and all major browsers.

Base64 comes from IETF \href{https://tools.ietf.org/html/rfc4648\#section-4}{\texttt{ RFC 4648}} (2006).


\begin{DoxyItemize}
\item {\bfseries{{\ttfamily btoa}}}, the encoder function, stands for {\bfseries{binary}} to {\bfseries{ASCII}}, meaning it converts any binary input into a subset of {\bfseries{ASCII}} (Base64).
\item {\bfseries{{\ttfamily atob}}}, the decoder function, converts {\bfseries{ASCII}} (or Base64) to its original {\bfseries{binary}} format. 
\end{DoxyItemize}
