{\bfseries{clone}} masters cloning simple objects (even with custom prototype), arrays, Date objects, and Reg\+Exp objects. Everything is cloned recursively, so that you can clone dates in arrays in objects, for example.\hypertarget{README.md_autotoc_md10584}{}\doxysubsection{\texorpdfstring{API}{API}}\label{README.md_autotoc_md10584}
{\ttfamily clone(val, circular, depth)}


\begin{DoxyItemize}
\item {\ttfamily val} -- the value that you want to clone, any type allowed
\item {\ttfamily circular} -- boolean

Call {\ttfamily clone} with {\ttfamily circular} set to {\ttfamily false} if you are certain that {\ttfamily obj} contains no circular references. This will give better performance if needed. There is no error if {\ttfamily undefined} or {\ttfamily null} is passed as {\ttfamily obj}.
\item {\ttfamily depth} -- depth to which the object is to be cloned (optional, defaults to infinity)
\end{DoxyItemize}

{\ttfamily clone.\+clone\+Prototype(obj)}


\begin{DoxyItemize}
\item {\ttfamily obj} -- the object that you want to clone
\end{DoxyItemize}

Does a prototype clone as \href{http://oranlooney.com/functional-javascript/}{\texttt{ described by Oran Looney}}.\hypertarget{README.md_autotoc_md10585}{}\doxysubsection{\texorpdfstring{Circular References}{Circular References}}\label{README.md_autotoc_md10585}
