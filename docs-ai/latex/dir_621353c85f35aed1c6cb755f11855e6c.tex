If none of the methods are supported, and the environment variable isn\textquotesingle{}t set, the default width value is going to be {\ttfamily 0}, that can be changed using the configurable {\ttfamily options}.\hypertarget{README.md_autotoc_md10504}{}\doxysubsection{\texorpdfstring{API}{API}}\label{README.md_autotoc_md10504}

{\ttfamily cli\+Width} can be configured using an {\ttfamily options} parameter, the possible properties are\+:


\begin{DoxyItemize}
\item {\bfseries{default\+Width}}\texorpdfstring{$<$}{<}number\texorpdfstring{$>$}{>} Defines a default value to be used if none of the methods are available, defaults to {\ttfamily 0}
\item {\bfseries{output}}\texorpdfstring{$<$}{<}object\texorpdfstring{$>$}{>} A stream to be used to read width values from, defaults to {\ttfamily process.\+stdout}
\item {\bfseries{tty}}\texorpdfstring{$<$}{<}object\texorpdfstring{$>$}{>} TTY module to try to read width from as a fallback, defaults to `require(\textquotesingle{}tty')\`{}
