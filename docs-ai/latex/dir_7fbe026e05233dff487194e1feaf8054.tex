\doxysubsection{Detailed Description}
Runtime type checking for React props and similar objects.

You can use prop-\/types to document the intended types of properties passed to components. React (and potentially other libraries—see the {\ttfamily check\+Prop\+Types()} reference below) will check props passed to your components against those definitions, and warn in development if they don’t match.\hypertarget{README.md_autotoc_md26632}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md26632}
