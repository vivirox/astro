\hypertarget{README.md_autotoc_md15219}{}\doxysubsection{\texorpdfstring{Main Features}{Main Features}}\label{README.md_autotoc_md15219}



\begin{DoxyItemize}
\item Validate XML data syntactically
\item Parse XML to JS Object
\item Build XML from JS Object
\item Compatible to node js packages, in browser, and in CLI (click try me button above for demo)
\item Faster than any other pure JS implementation.
\item It can handle big files (tested up to 100mb).
\item Controlled parsing using various options
\item XML Entities, HTML entities, and DOCTYPE entites are supported.
\item unpaired tags (Eg {\ttfamily \texorpdfstring{$<$}{<}br\texorpdfstring{$>$}{>}} in HTML), stop nodes (Eg {\ttfamily \texorpdfstring{$<$}{<}script\texorpdfstring{$>$}{>}} in HTML) are supported.
\item You can restore almost same XML from JSON
\item Supports comments
\item It can preserve Order of tags in JS object
\item You can control if a single tag should be parsed into array.
\item Supports parsing of PI (Processing Instruction) tags with XML declaration tags
\item And many more other features.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md15220}{}\doxysubsection{\texorpdfstring{v5}{v5}}\label{README.md_autotoc_md15220}
