\doxysubsection{Detailed Description}
Bytes go in, but they don\textquotesingle{}t come out (when muted).

This is a basic pass-\/through stream, but when muted, the bytes are silently dropped, rather than being passed through.\hypertarget{README.md_autotoc_md24416}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md24416}
