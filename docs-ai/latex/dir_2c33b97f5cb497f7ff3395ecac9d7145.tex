\doxysubsection{Detailed Description}
The core package of \href{https://expressive-code.com/}{\texttt{ Expressive Code}}, an engine for presenting source code on the web.\hypertarget{README.md_autotoc_md3293}{}\doxysubsection{\texorpdfstring{Documentation}{Documentation}}\label{README.md_autotoc_md3293}
\href{https://expressive-code.com/}{\texttt{ Read the Expressive Code docs}} to learn more about the features provided by Expressive Code.\hypertarget{README.md_autotoc_md3294}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{README.md_autotoc_md3294}
Using this core package directly is {\bfseries{only recommended for advanced use cases}}.

Unless you\textquotesingle{}re a plugin or integration author, you should probably use a higher-\/level package like \href{https://www.npmjs.com/package/astro-expressive-code}{\texttt{ {\ttfamily astro-\/expressive-\/code}}} or \href{https://www.npmjs.com/package/rehype-expressive-code}{\texttt{ {\ttfamily rehype-\/expressive-\/code}}} instead of this one.\hypertarget{README.md_autotoc_md3295}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md3295}
