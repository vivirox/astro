\doxysubsection{Detailed Description}
\begin{quote}
Super fast, memoized {\ttfamily req.\+url} parser; {\itshape not} limited to \href{https://github.com/lukeed/polka}{\texttt{ Polka}}! \end{quote}


Parses the {\ttfamily url} from a \href{https://nodejs.org/api/http.html\#http_class_http_incomingmessage}{\texttt{ {\ttfamily Incoming\+Message}}} request. The returned object will always only contain the following keys\+: {\ttfamily search}, {\ttfamily query}, {\ttfamily pathname}, and {\ttfamily raw}.

\begin{quote}
{\bfseries{Note\+:}} This library does not process {\ttfamily protocol}, {\ttfamily hostname}, {\ttfamily port}, etc.~\newline
This is because the incoming {\ttfamily req.\+url} value only begins with the path information. \end{quote}


Parsed requests will be mutated with a {\ttfamily \+\_\+parsed\+Url} key, containing the returned output. This is used for future memoization, avoiding the need to fully parse the same {\ttfamily url} value multiple times.\hypertarget{readme.md_autotoc_md5484}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md5484}
