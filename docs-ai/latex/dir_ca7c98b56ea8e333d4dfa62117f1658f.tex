\doxysubsection{Detailed Description}
This library is aim to explicitly annotate path format of the code, and throwing error when passing incorrect format paths.

The advantage is that we can avoid using {\ttfamily upath} to convert all paths to unix path, and ensure current code logic always correctly know the path format that should be processed.

Please note that you should only use it in your modules that are coupled to the filesystem.\hypertarget{README.md_autotoc_md33930}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md33930}
