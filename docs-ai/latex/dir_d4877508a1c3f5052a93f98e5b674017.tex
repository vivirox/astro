This package contains an extension to turn off some markdown constructs when parsing.\hypertarget{readme.md_autotoc_md23412}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md23412}
This project is useful when you want to disable support for code (indented), autolinks, and HTML (flow and text) in markdown.

You can use this extension when you are working with \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}}. To support all MDX features, use \href{https://github.com/micromark/micromark-extension-mdxjs}{\texttt{ {\ttfamily micromark-\/extension-\/mdxjs}}} instead.

All these packages are used in \href{https://mdxjs.com/packages/remark-mdx/}{\texttt{ {\ttfamily remark-\/mdx}}}, which focusses on making it easier to transform content by abstracting these internals away.\hypertarget{readme.md_autotoc_md23413}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md23413}
