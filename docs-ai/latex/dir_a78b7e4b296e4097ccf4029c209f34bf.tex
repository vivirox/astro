\doxysubsection{Detailed Description}
Like the unix {\ttfamily which} utility.

Finds the first instance of a specified executable in the PATH environment variable. Does not cache the results, so {\ttfamily hash -\/r} is not needed when the PATH changes.\hypertarget{README.md_autotoc_md35668}{}\doxysubsection{\texorpdfstring{USAGE}{USAGE}}\label{README.md_autotoc_md35668}
