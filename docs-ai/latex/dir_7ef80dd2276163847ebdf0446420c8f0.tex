\doxysubsection{Detailed Description}
\hypertarget{README.md_autotoc_md3204}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{README.md_autotoc_md3204}
This packages contains functions that allow you to wrap existing ESLint rules, plugins, and configurations that were intended for use with ESLint v8.\+x to allow them to work as-\/is in ESLint v9.\+x.

{\bfseries{Note\+:}} All plugins are not guaranteed to work in ESLint v9.\+x. This package fixes the most common issues but can\textquotesingle{}t fix everything.\hypertarget{README.md_autotoc_md3205}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md3205}

\hypertarget{README.md_autotoc_md3206}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md3206}
This package exports the following functions in both ESM and Common\+JS format\+:


\begin{DoxyItemize}
\item {\ttfamily fixup\+Rule(rule)} -\/ wraps the given rule in a compatibility layer and returns the result
\item {\ttfamily fixup\+Plugin\+Rules(plugin)} -\/ wraps each rule in the given plugin using {\ttfamily fixup\+Rule()} and returns a new object that represents the plugin with the fixed-\/up rules
\item {\ttfamily fixup\+Config\+Rules(configs)} -\/ wraps all plugins found in an array of config objects using {\ttfamily fixup\+Plugin\+Rules()}
\item {\ttfamily include\+Ignore\+File(path)} -\/ reads an ignore file (like {\ttfamily .gitignore}) and converts the patterns into the correct format for the config file
\end{DoxyItemize}\hypertarget{README.md_autotoc_md3207}{}\doxysubsubsection{\texorpdfstring{Fixing Rules}{Fixing Rules}}\label{README.md_autotoc_md3207}
