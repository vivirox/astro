\doxysubsection{Detailed Description}
\begin{quote}
Truncate a string to a specific width in the terminal \end{quote}


Gracefully handles \href{https://en.wikipedia.org/wiki/ANSI_escape_code\#Colors_and_Styles}{\texttt{ ANSI escapes}}. Like a string styled with \href{https://github.com/chalk/chalk}{\texttt{ {\ttfamily chalk}}}. It also supports Unicode surrogate pairs and fullwidth characters.\hypertarget{readme.md_autotoc_md10490}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md10490}
