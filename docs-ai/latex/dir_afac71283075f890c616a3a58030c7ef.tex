This ESLint plugin provides linting rules for \href{https://toml.io/}{\texttt{ TOML}}.


\begin{DoxyItemize}
\item You can use ESLint to lint \href{https://toml.io/}{\texttt{ TOML}}.
\item You can find out the problem with your \href{https://toml.io/}{\texttt{ TOML}} files.
\item You can apply consistent code styles to your \href{https://toml.io/}{\texttt{ TOML}} files.
\item Supports \href{https://vue-loader.vuejs.org/spec.html}{\texttt{ Vue SFC}} custom blocks such as {\ttfamily \texorpdfstring{$<$}{<}custom-\/block lang="{}toml"{}\texorpdfstring{$>$}{>}}. ~\newline
 Requirements {\ttfamily vue-\/eslint-\/parser} v7.\+3.\+0 and above.
\item Supports ESLint directives. e.\+g. {\ttfamily \# eslint-\/disable-\/next-\/line}
\item You can check your code in real-\/time using the ESLint editor integrations.
\end{DoxyItemize}
