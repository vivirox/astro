This plugin adds support to unified for parsing Latin-\/script natural language. If the prose is English or Dutch, use {\ttfamily unified} itself with \href{https://github.com/retextjs/retext/tree/main/packages/retext-stringify}{\texttt{ {\ttfamily retext-\/stringify}}} and either \href{https://github.com/retextjs/retext/tree/main/packages/retext-english}{\texttt{ {\ttfamily retext-\/english}}} or \href{https://github.com/retextjs/retext/tree/main/packages/retext-dutch}{\texttt{ {\ttfamily retext-\/dutch}}}. Or, use \href{https://github.com/retextjs/retext/tree/main/packages/retext}{\texttt{ {\ttfamily retext}}} directly.
