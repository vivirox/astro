\doxysubsection{Detailed Description}
This directory contains an integration with \href{https://github.com/Scarelette/MentalArena}{\texttt{ Mental\+Arena}}, a self-\/play framework for training language models for diagnosis and treatment of mental health disorders.\hypertarget{README.md_autotoc_md37396}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{README.md_autotoc_md37396}
Mental\+Arena enhances our Emotion\+LLa\+MA implementation by providing\+:


\begin{DoxyEnumerate}
\item {\bfseries{Self-\/play training}} -\/ Generates domain-\/specific personalized data through simulated patient-\/therapist interactions
\item {\bfseries{Symptom Encoder}} -\/ Simulates realistic patients from cognitive and behavioral perspectives
\item {\bfseries{Symptom Decoder}} -\/ Compares diagnosed symptoms with encoded symptoms to improve treatment accuracy
\item {\bfseries{Synthetic Data Generation}} -\/ Creates high-\/quality therapeutic conversations to improve model training
