\hypertarget{README.md_autotoc_md11590}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md11590}

\begin{DoxyItemize}
\item Integers and floats
\item Simple but full-\/featured API
\item Replicates many of the methods of Java\+Script\textquotesingle{}s {\ttfamily Number.\+prototype} and {\ttfamily Math} objects
\item Also handles hexadecimal, binary and octal values
\item Faster, smaller, and perhaps easier to use than Java\+Script versions of Java\textquotesingle{}s Big\+Decimal
\item No dependencies
\item Wide platform compatibility\+: uses Java\+Script 1.\+5 (ECMAScript 3) features only
\item Comprehensive \href{https://mikemcl.github.io/decimal.js/}{\texttt{ documentation}} and test set
\item Used under the hood by \href{https://github.com/josdejong/mathjs}{\texttt{ math.\+js}}
\item Includes a Type\+Script declaration file\+: {\itshape decimal.\+d.\+ts}
\end{DoxyItemize}



The library is similar to \href{https://github.com/MikeMcl/bignumber.js/}{\texttt{ bignumber.\+js}}, but here precision is specified in terms of significant digits rather than decimal places, and all calculations are rounded to the precision (similar to Python\textquotesingle{}s decimal module) rather than just those involving division.

This library also adds the trigonometric functions, among others, and supports non-\/integer powers, which makes it a significantly larger library than {\itshape bignumber.\+js} and the even smaller \href{https://github.com/MikeMcl/big.js/}{\texttt{ big.\+js}}.

For a lighter version of this library without the trigonometric functions see \href{https://github.com/MikeMcl/decimal.js-light/}{\texttt{ decimal.\+js-\/light}}.\hypertarget{README.md_autotoc_md11591}{}\doxysubsubsection{\texorpdfstring{Load}{Load}}\label{README.md_autotoc_md11591}

All calculations are rounded according to the number of significant digits and rounding mode specified by the {\ttfamily precision} and {\ttfamily rounding} properties of the Decimal constructor.
