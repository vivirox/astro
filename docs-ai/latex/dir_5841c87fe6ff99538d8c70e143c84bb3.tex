\hypertarget{README.md_autotoc_md22972}{}\doxysubsubsubsection{\texorpdfstring{The zoomed image is not visible}{The zoomed image is not visible}}\label{README.md_autotoc_md22972}
The library doesn\textquotesingle{}t provide a {\ttfamily z-\/index} value on the zoomed image to avoid conflicts with other frameworks. Some frameworks might specify a {\ttfamily z-\/index} for their elements, which makes the zoomed image not visible.

If that\textquotesingle{}s the case, you can provide a {\ttfamily z-\/index} value in your CSS\+:
