\doxysubsection{Detailed Description}
Generate a \href{https://github.com/vercel/satori}{\texttt{ satori}}-\/friendly VDOM from a string of HTML.\hypertarget{README.md_autotoc_md30248}{}\doxysubsection{\texorpdfstring{What is this?}{What is this?}}\label{README.md_autotoc_md30248}
\href{https://github.com/vercel/satori}{\texttt{ Satori}} is an amazing library for generating SVG strings from pure HTML and CSS.

Unfortunately, it is built on top of React\textquotesingle{}s JSX and \href{https://github.com/vercel/satori\#use-without-jsx}{\texttt{ expects "{}\+React-\/elements-\/like objects"{}}}. This library ({\ttfamily satori-\/html}) bridges that gap, generating the necessary VDOM object from a string of HTML.

\begin{quote}
{\bfseries{Note}} Satori supports a limited subset of HTML and CSS features, due to its special use case. Please use inline styles rather than class-\/based styling! \end{quote}

{\ttfamily satori-\/html} exports an {\ttfamily html} helper, which transforms HTML strings into an object that is compatible with {\ttfamily satori}.

\DoxyCodeLine{//\ See\ https://github.com/vercel/satori\#documentation}
\DoxyCodeLine{const\ svg\ =\ await\ satori(markup,\ \{}
\DoxyCodeLine{\ \ width:\ 600,}
\DoxyCodeLine{\ \ height:\ 400,}
\DoxyCodeLine{\ \ fonts:\ [],}
