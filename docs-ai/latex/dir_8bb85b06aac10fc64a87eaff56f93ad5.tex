\hypertarget{README.md_autotoc_md21483}{}\doxysubsubsection{\texorpdfstring{tags.\+search(description, \mbox{[}all\mbox{]})}{tags.\+search(description, \mbox{[}all\mbox{]})}}\label{README.md_autotoc_md21483}
Search for tags and subtags by description. Supports either a Reg\+Exp object or a string for {\ttfamily description}. Returns an array of {\ttfamily Subtag} and {\ttfamily Tag} objects or an empty array if no results were found.

Note that {\ttfamily Tag} objects in the results represent \textquotesingle{}grandfathered\textquotesingle{} or \textquotesingle{}redundant\textquotesingle{} tags. These are excluded by default. Set the {\ttfamily all} parameter to {\ttfamily true} to include them.

Search is case-\/insensitive if {\ttfamily description} is a string.\hypertarget{README.md_autotoc_md21484}{}\doxysubsubsection{\texorpdfstring{tags.\+languages(macrolanguage)}{tags.\+languages(macrolanguage)}}\label{README.md_autotoc_md21484}

Throws an error if {\ttfamily macrolanguage} is not a macrolanguage.

Return the subtag code formatted according to the case conventions defined in \href{http://tools.ietf.org/html/rfc5646\#section-2.1.1}{\texttt{ RFC 5646 section 2.\+1.\+1}}.

\item language codes are made lowercase (\textquotesingle{}mn\textquotesingle{} for Mongolian)
\item script codes are made lowercase with the initial letter capitalized (\textquotesingle{}Cyrl\textquotesingle{} for Cyrillic)
