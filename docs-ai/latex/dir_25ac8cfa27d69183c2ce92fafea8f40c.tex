\doxysubsection{Detailed Description}
\begin{quote}
Returns {\ttfamily true} if the given string looks like a glob pattern or an extglob pattern. This makes it easy to create code that only uses external modules like node-\/glob when necessary, resulting in much faster code execution and initialization time, and a better user experience. \end{quote}

\hypertarget{README.md_autotoc_md18610}{}\doxysubsubsection{\texorpdfstring{Option strict}{Option strict}}\label{README.md_autotoc_md18610}
When {\ttfamily options.\+strict === false} the behavior is less strict in determining if a pattern is a glob. Meaning that some patterns that would return {\ttfamily false} may return {\ttfamily true}. This is done so that matching libraries like \href{https://github.com/micromatch/micromatch}{\texttt{ micromatch}} have a chance at determining if the pattern is a glob or not.
