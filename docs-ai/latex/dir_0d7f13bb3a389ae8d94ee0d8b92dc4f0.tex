\item {\bfseries{highly customizable}} â€“ with a little code you can deeply customize how comments are parsed
\item {\bfseries{bidirectional}} -\/ you can write comment blocks back to the source after updating or formatting
\item {\bfseries{strictly typed}} -\/ comes with generated {\ttfamily d.\+ts} data definitions since written in Type\+Script
