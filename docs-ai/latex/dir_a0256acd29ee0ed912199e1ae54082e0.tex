This package is a small utility that checks that a node is a certain node.\hypertarget{readme.md_autotoc_md34264}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md34264}
Use this small utility if you find yourself repeating code for checking what nodes are.

A similar package, \href{https://github.com/syntax-tree/hast-util-is-element}{\texttt{ {\ttfamily hast-\/util-\/is-\/element}}}, works on hast elements.

For more advanced tests, \href{https://github.com/syntax-tree/unist-util-select}{\texttt{ {\ttfamily unist-\/util-\/select}}} can be used to match against CSS selectors.\hypertarget{readme.md_autotoc_md34265}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md34265}
