\doxysubsection{Detailed Description}
\begin{quote}
Trace the original position through a source map \end{quote}


{\ttfamily trace-\/mapping} allows you to take the line and column of an output file and trace it to the original location in the source file through a source map.

You may already be familiar with the \href{https://www.npmjs.com/package/source-map}{\texttt{ {\ttfamily source-\/map}}} package\textquotesingle{}s {\ttfamily Source\+Map\+Consumer}. This provides the same {\ttfamily original\+Position\+For} and {\ttfamily generated\+Position\+For} API, without requiring WASM.\hypertarget{README.md_autotoc_md4698}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md4698}
