\doxysubsection{Detailed Description}
Java\+Script library of crypto standards.\hypertarget{README.md_autotoc_md11086}{}\doxysubsection{\texorpdfstring{Discontinued}{Discontinued}}\label{README.md_autotoc_md11086}
Active development of Crypto\+JS has been discontinued. This library is no longer maintained.

Nowadays, Node\+JS and modern browsers have a native {\ttfamily Crypto} module. The latest version of Crypto\+JS already uses the native Crypto module for random number generation, since {\ttfamily Math.\+random()} is not crypto-\/safe. Further development of Crypto\+JS would result in it only being a wrapper of native Crypto. Therefore, development and maintenance has been discontinued, it is time to go for the native {\ttfamily crypto} module.\hypertarget{README.md_autotoc_md11087}{}\doxysubsection{\texorpdfstring{Node.\+js (Install)}{Node.\+js (Install)}}\label{README.md_autotoc_md11087}
