CAC, or cac, pronounced {\ttfamily C-\/\+A-\/C}.

This project is dedicated to our lovely C.\+C. sama. Maybe CAC stands for C\&C as well \+:P

\hypertarget{README.md_autotoc_md9739}{}\doxysubsubsubsection{\texorpdfstring{Why not use Commander.\+js?}{Why not use Commander.\+js?}}\label{README.md_autotoc_md9739}
CAC is very similar to Commander.\+js, while the latter does not support dot nested options, i.\+e. something like {\ttfamily -\/-\/env.\+API\+\_\+\+SECRET foo}. Besides, you can\textquotesingle{}t use unknown options in Commander.\+js either.
