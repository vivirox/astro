This package is a utility that can turn a hast syntax tree into a {\ttfamily parse5} AST. Why not use a Parse5 adapter, you might ask? Well, because it’s more code weight to use adapters, and more fragile.\hypertarget{readme.md_autotoc_md17540}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md17540}
This package is useful when working with {\ttfamily parse5}, and for some reason want to generate its AST again. The inverse utility, \href{https://github.com/syntax-tree/hast-util-from-parse5}{\texttt{ {\ttfamily hast-\/util-\/from-\/parse5}}}, is more likely what you want.\hypertarget{readme.md_autotoc_md17541}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md17541}
