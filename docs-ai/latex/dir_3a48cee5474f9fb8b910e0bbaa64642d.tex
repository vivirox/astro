To customize formatting behavior, see the \doxylink{README.md_configuration}{Configuration} section below.\hypertarget{README.md_autotoc_md26311}{}\doxysubsection{\texorpdfstring{Formatting with the VS Code Prettier extension directly}{Formatting with the VS Code Prettier extension directly}}\label{README.md_autotoc_md26311}
\begin{quote}
{\bfseries{Note}} \href{https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode}{\texttt{ The Astro VS Code extension}} uses Prettier and this plugin ({\ttfamily prettier-\/plugin-\/astro}) to format your code. You will only need to install the VS Code Prettier extension separately for formatting if\+: \end{quote}



\begin{DoxyItemize}
\item You are not using Astro\textquotesingle{}s VS Code extension.
\item You want to use features of the Prettier extension that not supported by Astro\textquotesingle{}s own VS Code extension, such as the toolbar panel showing Prettier\textquotesingle{}s status.
\end{DoxyItemize}

Install the \href{https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode}{\texttt{ VS Code Prettier extension}} and add the following settings to your VS Code configuration\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ "{}prettier.documentSelectors"{}:\ ["{}**/*.astro"{}],}
\DoxyCodeLine{\ \ "{}[astro]"{}:\ \{}
\DoxyCodeLine{\ \ \ \ "{}editor.defaultFormatter"{}:\ "{}esbenp.prettier-\/vscode"{}}
\DoxyCodeLine{\ \ \}}
\DoxyCodeLine{\}}

\end{DoxyCode}
