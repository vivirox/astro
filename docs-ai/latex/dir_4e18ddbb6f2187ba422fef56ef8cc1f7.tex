This package is a utility that formats whitespace in HTML. In short, it works as follows\+:


\begin{DoxyItemize}
\item collapse all existing whitespace to either a line ending or a single space (\href{https://github.com/rehypejs/rehype-minify/tree/main/packages/hast-util-minify-whitespace}{\texttt{ {\ttfamily hast-\/util-\/minify-\/whitespace}}})
\item remove those spaces and line endings if they do not contribute to the document
\item inject needed line endings
\item indent previously collapsed line endings properly
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md16549}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md16549}
This package is useful when you want to improve the readability of an HTML fragment as it adds insignificant but pretty whitespace between elements. The plugin \href{https://github.com/rehypejs/rehype-format}{\texttt{ {\ttfamily rehype-\/format}}} uses this package to format HTML at the plugin level. The package \href{https://github.com/rehypejs/rehype-minify/tree/main/packages/hast-util-minify-whitespace}{\texttt{ {\ttfamily hast-\/util-\/minify-\/whitespace}}} does the inverse.\hypertarget{readme.md_autotoc_md16550}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md16550}
