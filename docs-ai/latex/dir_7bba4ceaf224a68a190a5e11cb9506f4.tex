Normalize an identifier (as found in references, definitions).

Collapses markdown whitespace, trim, and then lower-\/ and uppercase.

Some characters are considered “uppercase”, such as U+03\+F4 ({\ttfamily ϴ}), but if their lowercase counterpart (U+03\+B8 ({\ttfamily θ})) is uppercased will result in a different uppercase character (U+0398 ({\ttfamily Θ})). So, to get a canonical form, we perform both lower-\/ and uppercase.

Using uppercase last makes sure keys will never interact with default prototypal values (such as {\ttfamily constructor})\+: nothing in the prototype of {\ttfamily Object} is uppercase.\hypertarget{readme.md_autotoc_md23748}{}\doxysubsubsubsubsubsection{\texorpdfstring{Parameters}{Parameters}}\label{readme.md_autotoc_md23748}
