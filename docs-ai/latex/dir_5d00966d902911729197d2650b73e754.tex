This package is a \href{https://github.com/unifiedjs/unified}{\texttt{ unified}} (\href{https://github.com/retextjs/retext}{\texttt{ retext}}) plugin to apply \href{https://daringfireball.net/projects/smartypants}{\texttt{ Smarty\+Pants}} to the syntax tree. It replaces straight/typewriter punctuation marks and symbols with smart/curly marks and symbols.\hypertarget{readme.md_autotoc_md29487}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md29487}
