\doxysubsection{Detailed Description}
fastfile is a package to read/write binary files with a transparent cache.

This library maintains a read and write cache that allows to write asynchronously improving the speed of the reads and writes of big files.\hypertarget{README.md_autotoc_md15234}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md15234}
