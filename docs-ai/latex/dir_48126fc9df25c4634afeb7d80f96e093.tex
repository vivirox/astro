The {\ttfamily Mapping} is a tuple that represents a mapping in the source map. It consists of the following elements\+:


\begin{DoxyItemize}
\item {\ttfamily source}\+: A string representing the source file. This can be {\ttfamily undefined}.
\item {\ttfamily source\+Offsets}\+: Offsets in the source code.
\item {\ttfamily generated\+Offsets}\+: Offsets in the generated code.
\item {\ttfamily data}\+: The data associated with this mapping. The type of this data is generic and can be specified when creating a {\ttfamily Source\+Map} instance.
\end{DoxyItemize}

Here is an example of a {\ttfamily Mapping}\+:

\DoxyCodeLine{\ \ \ \ data:\ myData,}
