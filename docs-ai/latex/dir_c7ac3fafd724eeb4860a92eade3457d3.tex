\doxysubsection{Detailed Description}
Basis Universal is a "{}\mbox{[}supercompressed\mbox{]}(http\+://gamma.\+cs.\+unc.\+edu/\+GST/gst.\+pdf)"{} GPU texture and texture video compression system that outputs a highly compressed intermediate file format (.basis) that can be quickly transcoded to a wide variety of GPU texture compression formats.

\href{https://github.com/BinomialLLC/basis_universal}{\texttt{ Git\+Hub}}\hypertarget{README.md_autotoc_md32804}{}\doxysubsection{\texorpdfstring{Transcoders}{Transcoders}}\label{README.md_autotoc_md32804}
Basis Universal texture data may be used in two different file formats\+: {\ttfamily .basis} and {\ttfamily .ktx2}, where {\ttfamily ktx2} is a standardized wrapper around basis texture data.

For further documentation about the Basis compressor and transcoder, refer to the \href{https://github.com/BinomialLLC/basis_universal}{\texttt{ Basis Git\+Hub repository}}.

The folder contains two files required for transcoding {\ttfamily .basis} or {\ttfamily .ktx2} textures\+:


\begin{DoxyItemize}
\item {\ttfamily basis\+\_\+transcoder.\+js} — Java\+Script wrapper for the Web\+Assembly transcoder.
\item {\ttfamily basis\+\_\+transcoder.\+wasm} — Web\+Assembly transcoder.
\end{DoxyItemize}
