\doxysubsection{Detailed Description}
An optional plugin for \href{https://expressive-code.com/}{\texttt{ Expressive Code}}, an engine for presenting source code on the web.

It allows code sections to be marked as collapsed. The lines in collapsed sections will be hidden by default, replaced by a "{}\+X collapsed lines"{} line. When clicked, the collapsed section will be expanded, showing the previously hidden lines.\hypertarget{README.md_autotoc_md3298}{}\doxysubsection{\texorpdfstring{Documentation}{Documentation}}\label{README.md_autotoc_md3298}
