In VS Code, there are two ways of loading custom CSS datasets\+:


\begin{DoxyEnumerate}
\item With setting {\ttfamily css.\+custom\+Data} 
\begin{DoxyCode}{0}
\DoxyCodeLine{"{}css.customData"{}:\ [}
\DoxyCodeLine{\ \ \ \ "{}./foo.css-\/data.json"{}}

\item With an extension that contributes {\ttfamily contributes.\+css.\+custom\+Data}

Both setting point to a list of JSON files. This document describes the shape of the JSON files.

\begin{DoxyItemize}
\item {\ttfamily browsers}\+: A list of supported browsers. The format is {\ttfamily browser\+Name + version}. For example\+: `\mbox{[}\textquotesingle{}E10', \textquotesingle{}C30\textquotesingle{}, \textquotesingle{}FF20\textquotesingle{}\mbox{]}`. Here are all browser names\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{export\ let\ browserNames\ =\ \{}
\DoxyCodeLine{\ \ E:\ 'Edge\&rsquo;,}
\DoxyCodeLine{\ \ FF:\ 'Firefox',}
\DoxyCodeLine{\ \ S:\ 'Safari',}
\DoxyCodeLine{\ \ C:\ 'Chrome',}
\DoxyCodeLine{\ \ IE:\ 'IE',}
\DoxyCodeLine{\ \ O:\ 'Opera'}
\DoxyCodeLine{\};}

\end{DoxyCode}
 The browser compatibility will be rendered at completion and hover. Items that is supported in only one browser are dropped from completion.
\item {\ttfamily status}\+: The status of the item. The format is\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{export\ type\ EntryStatus\ =\ 'standard'\ |\ 'experimental'\ |\ 'nonstandard'\ |\ 'obsolete';}

\end{DoxyCode}
 The status will be rendered at the top of completion and hover. For example, {\ttfamily nonstandard} items are prefixed with the message {\ttfamily üö®Ô∏è Property is nonstandard. Avoid using it.}.
\item {\ttfamily references}\+: A list of references. They will be displayed in Markdown form in completion and hover as {\ttfamily \mbox{[}Ref1 Name\mbox{]}(Ref1 URL) \texorpdfstring{$\vert$}{|} \mbox{[}Ref2 Name\mbox{]}(Ref2 URL) \texorpdfstring{$\vert$}{|} ...}.
\item {\ttfamily relevance}\+: A number in the range \mbox{[}0, 100\mbox{]} used for sorting. Bigger number means more relevant and will be sorted first. Entries that do not specify a relevance will get 50 as default value. 
\end{DoxyItemize}
