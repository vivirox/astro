This package contains two extensions that add support for GFM table syntax in markdown to \href{https://github.com/syntax-tree/mdast}{\texttt{ mdast}}. These extensions plug into \href{https://github.com/syntax-tree/mdast-util-from-markdown}{\texttt{ {\ttfamily mdast-\/util-\/from-\/markdown}}} (to support parsing tables in markdown into a syntax tree) and \href{https://github.com/syntax-tree/mdast-util-to-markdown}{\texttt{ {\ttfamily mdast-\/util-\/to-\/markdown}}} (to support serializing tables in syntax trees to markdown).\hypertarget{readme.md_autotoc_md22451}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md22451}
You can use these extensions when you are working with {\ttfamily mdast-\/util-\/from-\/markdown} and {\ttfamily mdast-\/util-\/to-\/markdown} already.

When working with {\ttfamily mdast-\/util-\/from-\/markdown}, you must combine this package with \href{https://github.com/micromark/micromark-extension-gfm-table}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/table}}}.

When you don‚Äôt need a syntax tree, you can use \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}} directly with {\ttfamily micromark-\/extension-\/gfm-\/table}.

When you are working with syntax trees and want all of GFM, use \href{https://github.com/syntax-tree/mdast-util-gfm}{\texttt{ {\ttfamily mdast-\/util-\/gfm}}} instead.

All these packages are used \href{https://github.com/remarkjs/remark-gfm}{\texttt{ {\ttfamily remark-\/gfm}}}, which focusses on making it easier to transform content by abstracting these internals away.

This utility does not handle how markdown is turned to HTML. That‚Äôs done by \href{https://github.com/syntax-tree/mdast-util-to-hast}{\texttt{ {\ttfamily mdast-\/util-\/to-\/hast}}}.\hypertarget{readme.md_autotoc_md22452}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22452}

An {\ttfamily align} field can be present. If present, it must be a list of {\bfseries{align\+Types}}. It represents how cells in columns are aligned.
