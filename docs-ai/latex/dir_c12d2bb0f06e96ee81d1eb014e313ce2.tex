\doxysubsection*{Denque}



\href{https://www.npmjs.com/package/denque}{\texttt{ }} \href{https://www.npmjs.com/package/denque}{\texttt{ }} \href{https://github.com/invertase/denque/actions/workflows/testing.yam}{\texttt{ }} \href{https://codecov.io/gh/invertase/denque}{\texttt{ }} \href{/LICENSE}{\texttt{ }} \href{https://twitter.com/invertaseio}{\texttt{ }} 

Denque is a well tested, extremely fast and lightweight \href{http://en.wikipedia.org/wiki/Double-ended_queue}{\texttt{ double-\/ended queue}} implementation with zero dependencies and includes Type\+Script types.

Double-\/ended queues can also be used as a\+:


\begin{DoxyItemize}
\item \href{http://en.wikipedia.org/wiki/Stack_\\(abstract_data_type\\)}{\texttt{ Stack}}
\item \href{http://en.wikipedia.org/wiki/Queue_\\(data_structure\\)}{\texttt{ Queue}}
\end{DoxyItemize}

This implementation is currently the fastest available, even faster than {\ttfamily double-\/ended-\/queue}, see the \href{https://docs.page/invertase/denque/benchmarks}{\texttt{ benchmarks}}.

Every queue operation is done at a constant {\ttfamily O(1)} -\/ including random access from {\ttfamily .peek\+At(index)}.
