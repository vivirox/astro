\begin{DoxyItemize}
\item Install/update the dependencies\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install}

\end{DoxyCode}

\item Make sure the code lints\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ run\ lint}

\end{DoxyCode}

\item Make sure all of the tests pass\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ test}

\end{DoxyCode}

\item Adhere to the coding conventions that are used elsewhere in the codebase.
\item New code must have new unit tests.
\item New features or changes to existing features must be documented in the \href{https://gitlab.com/philbooth/bfj/blob/master/README.md}{\texttt{ readme}} file.
\item Add yourself to the \href{https://gitlab.com/philbooth/bfj/blob/master/AUTHORS}{\texttt{ authors}} file.
\item Feel free to \href{https://gitlab.com/philbooth/bfj/issues/new}{\texttt{ open an issue}} first, if the change is one that you think needs some discussion. 
\end{DoxyItemize}
