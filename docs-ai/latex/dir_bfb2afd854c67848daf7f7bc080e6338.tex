\doxysubsection{Detailed Description}
\href{https://github.com/wooorm/lowlight/actions}{\texttt{ }} \href{https://codecov.io/github/wooorm/lowlight}{\texttt{ }} \href{https://www.npmjs.com/package/lowlight}{\texttt{ }} \href{https://bundlephobia.com/result?p=lowlight}{\texttt{ }}

Virtual syntax highlighting for virtual DOMs and non-\/\+HTML things, with language auto-\/detection. Perfect for \href{https://facebook.github.io/react/}{\texttt{ React}}, \href{https://github.com/Matt-Esch/virtual-dom}{\texttt{ VDOM}}, and others.

Lowlight is built to work with all syntaxes supported by \href{https://github.com/highlightjs/highlight.js}{\texttt{ highlight.\+js}}, that’s \href{https://github.com/highlightjs/highlight.js/blob/main/SUPPORTED_LANGUAGES.md}{\texttt{ 191 languages}} (and all 94 themes).

Want to use \href{https://github.com/PrismJS/prism}{\texttt{ Prism}} instead? Try \href{https://github.com/wooorm/refractor}{\texttt{ {\ttfamily refractor}}}!\hypertarget{readme.md_autotoc_md22086}{}\doxysubsection{\texorpdfstring{Contents}{Contents}}\label{readme.md_autotoc_md22086}

\hypertarget{readme.md_autotoc_md22109}{}\doxysubsection{\texorpdfstring{Browser}{Browser}}\label{readme.md_autotoc_md22109}
It is not suggested to use the pre-\/built files or requiring {\ttfamily lowlight} in the browser as that would include 916kB (260kB GZipped) of code.

Instead, require {\ttfamily lowlight/lib/core}, and include only the used highlighters. For example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ low\ =\ require('lowlight/lib/core')}
\DoxyCodeLine{var\ js\ =\ require('highlight.js/lib/languages/javascript')}
\DoxyCodeLine{}
\DoxyCodeLine{low.registerLanguage('javascript',\ js)}
\DoxyCodeLine{}
\DoxyCodeLine{low.highlight('js',\ '"{}use\ strict"{};')}
\DoxyCodeLine{//\ See\ \`{}Usage`\ for\ the\ results.}

\end{DoxyCode}


…when using \href{https://github.com/browserify/browserify}{\texttt{ browserify}} and minifying with \href{https://github.com/browserify/tinyify}{\texttt{ tinyify}} this results in 24kB of code (9kB with GZip).\hypertarget{readme.md_autotoc_md22110}{}\doxysubsection{\texorpdfstring{Related}{Related}}\label{readme.md_autotoc_md22110}
