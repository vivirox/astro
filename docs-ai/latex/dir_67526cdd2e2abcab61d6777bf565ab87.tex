Make a value safe for injection as a URL.

This encodes unsafe characters with percent-\/encoding and skips already encoded sequences (see \`{}normalize\+Uri\`{}). Further unsafe characters are encoded as character references (see \href{https://github.com/micromark/micromark/tree/main/packages/micromark-util-encode}{\texttt{ {\ttfamily micromark-\/util-\/encode}}}).

A regex of allowed protocols can be given, in which case the URL is sanitized. For example, {\ttfamily /\texorpdfstring{$^\wedge$}{\string^}(https?\texorpdfstring{$\vert$}{|}ircs?\texorpdfstring{$\vert$}{|}mailto\texorpdfstring{$\vert$}{|}xmpp)\$/i} can be used for {\ttfamily a\mbox{[}href\mbox{]}}, or {\ttfamily /\texorpdfstring{$^\wedge$}{\string^}https?\$/i} for {\ttfamily img\mbox{[}src\mbox{]}} (this is what {\ttfamily github.\+com} allows). If the URL includes an unknown protocol (one not matched by {\ttfamily protocol}, such as a dangerous example, {\ttfamily javascript\+:}), the value is ignored.\hypertarget{readme.md_autotoc_md23781}{}\doxysubsubsubsubsubsection{\texorpdfstring{Parameters}{Parameters}}\label{readme.md_autotoc_md23781}
