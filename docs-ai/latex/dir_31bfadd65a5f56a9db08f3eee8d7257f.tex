\doxysubsection{Detailed Description}
Compact single select prompt. Every option is assigned a shortcut key, and selecting {\ttfamily h} will expand all the choices and their descriptions.

{\ttfamily Separator} objects can be used in the {\ttfamily choices} array to render non-\/selectable lines in the choice list. By default it\textquotesingle{}ll render a line, but you can provide the text as argument (`new Separator('-- Dependencies --\textquotesingle{})\`{}). This option is often used to add labels to groups within long list of options.\hypertarget{README.md_autotoc_md3750}{}\doxysubsubsubsection{\texorpdfstring{{\ttfamily Choice} object}{{\ttfamily Choice} object}}\label{README.md_autotoc_md3750}

Here\textquotesingle{}s each property\+:


\begin{DoxyItemize}
\item {\ttfamily value}\+: The value is what will be returned by {\ttfamily await expand()}.
\item {\ttfamily name}\+: The string displayed in the choice list. It\textquotesingle{}ll default to the stringify {\ttfamily value}.
\item {\ttfamily key}\+: The input the use must provide to select the choice. Must be a lowercase single alpha-\/numeric character string.
