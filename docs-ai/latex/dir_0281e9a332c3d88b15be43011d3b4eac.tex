\doxysubsection{Detailed Description}
A default plugin of Expressive Code, an engine for presenting source code on the web.

It renders a window frame around every code block. Depending on the code\textquotesingle{}s language, this frame can look like a code editor (similar to VS Code), or like a terminal window.\hypertarget{README.md_autotoc_md3301}{}\doxysubsection{\texorpdfstring{Documentation}{Documentation}}\label{README.md_autotoc_md3301}
\href{https://expressive-code.com/key-features/frames/}{\texttt{ Read this plugin\textquotesingle{}s documentation}} on the Expressive Code website to learn more about its features.\hypertarget{README.md_autotoc_md3302}{}\doxysubsection{\texorpdfstring{Installation (not required)}{Installation (not required)}}\label{README.md_autotoc_md3302}
