This package is a \href{https://github.com/unifiedjs/unified}{\texttt{ unified}} (\href{https://github.com/mdx-js/recma}{\texttt{ recma}}) that can turn JSX into Java\+Script.\hypertarget{readme.md_autotoc_md27939}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md27939}
Use this when preparing JSX for evaluation. JSX cannot be evaluated without being transformed first.

If you don’t use plugins and access syntax trees manually, you can directly use \href{https://github.com/syntax-tree/estree-util-build-jsx}{\texttt{ {\ttfamily estree-\/util-\/build-\/jsx}}}, which is used inside this plugin. recma focusses on making it easier to transform code by abstracting such internals away.\hypertarget{readme.md_autotoc_md27940}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md27940}
