\item {\ttfamily async\+: boolean}\+: By default, this plugin uses {\ttfamily read\+File\+Sync} to read the contents of the files. Set this to {\ttfamily true} if you want to use {\ttfamily read\+File} for non-\/blocking IO.
\item {\ttfamily root\+Dir\+: string}\+: Change what {\ttfamily \texorpdfstring{$<$}{<}root\+Dir\texorpdfstring{$>$}{>}} refers to. Defaults to {\ttfamily process.\+cwd()}.
\item {\ttfamily preserve\+Trailing\+Newline\+: boolean}\+: By default, this plugin will trim the trailing newline of the file when importing the code. You can preserve the trailing new line in the code block by setting this option to {\ttfamily true}.
\item {\ttfamily remove\+Redundant\+Indentations\+: boolean}\+: Set to {\ttfamily true} to remove redundant indentations for each line. For instance, the imported code of\+:
