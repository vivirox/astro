\href{https://www.typescriptlang.org/}{\texttt{ Type\+Script}} is a language for application-\/scale Java\+Script. Type\+Script adds optional types to Java\+Script that support tools for large-\/scale Java\+Script applications for any browser, for any host, on any OS.
