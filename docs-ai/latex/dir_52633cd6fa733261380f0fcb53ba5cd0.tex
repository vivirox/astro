\doxysubsection{Detailed Description}
\begin{quote}
A tiny (203B) and fast utility to flatten an object with customizable glue \end{quote}


This module recursively squashes an Object/\+Array. The output is a flat object – AKA, it has a single level of depth.

By default, the {\ttfamily .} character is used to glue/join layers\textquotesingle{} keys together. This is customizable.

Finally, by default, any keys with nullish values ({\ttfamily null} and {\ttfamily undefined}) are {\bfseries{not}} included in the return object.\hypertarget{readme.md_autotoc_md15628}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md15628}
