\href{https://medium.com}{\texttt{ Medium}}\textquotesingle{}s like reading time estimation.

{\ttfamily reading-\/time} helps you estimate how long an article will take to read. It works perfectly with plain text, but also with {\ttfamily markdown} or {\ttfamily html}.

Note that it\textquotesingle{}s focused on performance and simplicity, so the number of words it will extract from other formats than plain text can vary a little.
