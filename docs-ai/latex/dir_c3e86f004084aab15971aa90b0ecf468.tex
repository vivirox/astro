This package is a utility to get the plain-\/text value of a node.\hypertarget{readme.md_autotoc_md17560}{}\doxysubsubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md17560}
You can use this package when you want to get the plain text value of a node. The algorithm used by this package is like the DOMs {\ttfamily Node\#text\+Content} getter.
