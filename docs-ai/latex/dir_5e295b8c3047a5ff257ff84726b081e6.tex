\doxysubsection{Detailed Description}
\href{http://opendefinition.org/}{\texttt{ }}

IANA\textquotesingle{}s \href{http://www.iana.org/assignments/language-subtag-registry/language-subtag-registry}{\texttt{ official repository}} is in \href{http://www.inter-locale.com/ID/draft-phillips-record-jar-01.html}{\texttt{ record-\/jar}} format and is hard to parse. This project provides neatly organized JSON files representing that data.

See {\ttfamily data/} for all the JSON files available. The {\ttfamily registry.\+json} file contains all records in a flat array and {\ttfamily meta.\+json} contains its metadata. There\textquotesingle{}s a separate JSON file for each \textquotesingle{}scope\textquotesingle{} (e.\+g. {\ttfamily macrolanguage.\+json}) and \textquotesingle{}type\textquotesingle{} (e.\+g. {\ttfamily language.\+json}). These files contain JSON objects keyed by tag or subtag and with the index integer for the corresponding entry in {\ttfamily registry.\+json} as a value. This makes lookups fast.\hypertarget{README.md_autotoc_md21473}{}\doxysubsection{\texorpdfstring{Updates}{Updates}}\label{README.md_autotoc_md21473}
This project will be updated as the registry changes. Non-\/breaking updates will result in the patch version number being bumped.

Run {\ttfamily make update} to force an update from the latest official IANA-\/hosted version. The registry file format is converted to JSON automatically and the files in {\ttfamily data/} are updated.

If there are changes, please make a pull request.\hypertarget{README.md_autotoc_md21474}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md21474}
