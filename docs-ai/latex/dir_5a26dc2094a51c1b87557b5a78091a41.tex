\hypertarget{README.md_autotoc_md31229}{}\doxysubsection{\texorpdfstring{Call\+Site object}{Call\+Site object}}\label{README.md_autotoc_md31229}
From the \href{https://code.google.com/p/v8/wiki/JavaScriptStackTraceApi}{\texttt{ V8 Stack\+Trace API}}

The structured stack trace is an Array of Call\+Site objects, each of which represents a stack frame. A Call\+Site object defines the following methods

get\+This\+: returns the value of this ~\newline
 get\+Type\+Name\+: returns the type of this as a string. This is the name of the function stored in the constructor field of this, if available, otherwise the object\textquotesingle{}s \mbox{[}\mbox{[}Class\mbox{]}\mbox{]} internal property. ~\newline
 get\+Function\+: returns the current function ~\newline
 get\+Function\+Name\+: returns the name of the current function, typically its name property. If a name property is not available an attempt will be made to try to infer a name from the function\textquotesingle{}s context. ~\newline
 get\+Method\+Name\+: returns the name of the property of this or one of its prototypes that holds the current function ~\newline
 get\+File\+Name\+: if this function was defined in a script returns the name of the script ~\newline
 get\+Line\+Number\+: if this function was defined in a script returns the current line number ~\newline
 get\+Column\+Number\+: if this function was defined in a script returns the current column number ~\newline
 get\+Eval\+Origin\+: if this function was created using a call to eval returns a Call\+Site object representing the location where eval was called ~\newline
 is\+Toplevel\+: is this a toplevel invocation, that is, is this the global object? ~\newline
 is\+Eval\+: does this call take place in code defined by a call to eval? ~\newline
 is\+Native\+: is this call in native V8 code? ~\newline
 is\+Constructor\+: is this a constructor call? ~\newline
