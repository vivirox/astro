p5.\+js is a Java\+Script library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.\+js is free and open-\/source because we believe software, and the tools to learn it, should be accessible to everyone.

Using the metaphor of a sketch, p5.\+js has a full set of drawing functionality using the HTML5 canvas element. You‚Äôre not limited to the drawing canvas though. You can think of your whole browser page as your sketch, including HTML5 objects for text, input, video, webcam, and sound.

p5.\+js draws inspiration, wisdom, and guidance from its precursor \href{https://processing.org}{\texttt{ Processing}}. However, it is a new interpretation, not an emulation or port. We don\textquotesingle{}t aim to match Processing\textquotesingle{}s set of functionality exactly, allowing ourselves space to deviate and grow in the context of the web.

The \href{https://p5js.org}{\texttt{ p5js.\+org}} website contains an extensive overview of the project, community, documentation, and examples.
