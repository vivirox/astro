This package contains two extensions that add support for MDX syntax in markdown to \href{https://github.com/syntax-tree/mdast}{\texttt{ mdast}}\+: ESM (`import x from \textquotesingle{}y'{\ttfamily ), JSX (}{\ttfamily ), and expressions (}\{Math.\+PI\}{\ttfamily ). These extensions plug into \mbox{[}}mdast-\/util-\/from-\/markdown{\ttfamily \mbox{]}\mbox{[}mdast-\/util-\/from-\/markdown\mbox{]} (to support parsing MDX in markdown into a syntax tree) and \mbox{[}}mdast-\/util-\/to-\/markdown\`{}\mbox{]}\mbox{[}mdast-\/util-\/to-\/markdown\mbox{]} (to support serializing MDX in syntax trees to markdown).\hypertarget{readme.md_autotoc_md22639}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md22639}
You can use these extensions when you are working with {\ttfamily mdast-\/util-\/from-\/markdown} and {\ttfamily mdast-\/util-\/to-\/markdown} already.

When working with {\ttfamily mdast-\/util-\/from-\/markdown}, you must combine this package with \href{https://github.com/micromark/micromark-extension-mdx}{\texttt{ {\ttfamily micromark-\/extension-\/mdx}}} or \href{https://github.com/micromark/micromark-extension-mdxjs}{\texttt{ {\ttfamily micromark-\/extension-\/mdxjs}}}.

Instead of this package, you can also use the extensions separately\+:


\begin{DoxyItemize}
\item \href{https://github.com/syntax-tree/mdast-util-mdx-expression}{\texttt{ {\ttfamily mdast-\/util-\/mdx-\/expression}}} — support MDX expressions
\item \href{https://github.com/syntax-tree/mdast-util-mdx-jsx}{\texttt{ {\ttfamily mdast-\/util-\/mdx-\/jsx}}} — support MDX JSX
\item \href{https://github.com/syntax-tree/mdast-util-mdxjs-esm}{\texttt{ {\ttfamily mdast-\/util-\/mdxjs-\/esm}}} — support MDX ESM
\end{DoxyItemize}

All these packages are used in \href{https://github.com/mdx-js/mdx/tree/next/packages/remark-mdx}{\texttt{ {\ttfamily remark-\/mdx}}}, which focusses on making it easier to transform content by abstracting these internals away.\hypertarget{readme.md_autotoc_md22640}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22640}
