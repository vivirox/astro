Chai is an {\itshape assertion library}, similar to Node\textquotesingle{}s built-\/in {\ttfamily assert}. It makes testing much easier by giving you lots of assertions you can run against your code.\hypertarget{README.md_autotoc_md9921}{}\doxysubsubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md9921}

Chai offers a robust Plugin architecture for extending Chai\textquotesingle{}s assertions and interfaces.


\begin{DoxyItemize}
\item Need a plugin? View the \href{http://chaijs.com/plugins}{\texttt{ official plugin list}}.
\item Want to build a plugin? Read the \href{http://chaijs.com/guide/plugins/}{\texttt{ plugin api documentation}}.
\item Have a plugin and want it listed? Simply add the following keywords to your package.\+json\+:
\begin{DoxyItemize}
\item {\ttfamily chai-\/plugin}
\item {\ttfamily browser} if your plugin works in the browser as well as Node.\+js
\item {\ttfamily browser-\/only} if your plugin does not work with Node.\+js
\end{DoxyItemize}
