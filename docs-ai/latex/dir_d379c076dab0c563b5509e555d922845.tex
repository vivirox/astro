\doxysubsection{Detailed Description}
\begin{quote}
Get the shortest leading whitespace from lines in a string \end{quote}


The line with the least number of leading whitespace, ignoring empty lines, determines the number.

Useful for removing redundant indentation.\hypertarget{readme.md_autotoc_md24124}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md24124}
