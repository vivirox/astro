The {\ttfamily path\+To\+Regexp} function will return a regular expression object based on the provided {\ttfamily path} argument. It accepts the following arguments\+:


\begin{DoxyItemize}
\item {\bfseries{path}} A string, array of strings, or a regular expression.
\item {\bfseries{keys}} {\itshape (optional)} An array to populate with keys found in the path.
\item {\bfseries{options}} {\itshape (optional)}
\begin{DoxyItemize}
\item {\bfseries{sensitive}} When {\ttfamily true} the regexp will be case sensitive. (default\+: {\ttfamily false})
\item {\bfseries{strict}} When {\ttfamily true} the regexp won\textquotesingle{}t allow an optional trailing delimiter to match. (default\+: {\ttfamily false})
\item {\bfseries{end}} When {\ttfamily true} the regexp will match to the end of the string. (default\+: {\ttfamily true})
\item {\bfseries{start}} When {\ttfamily true} the regexp will match from the beginning of the string. (default\+: {\ttfamily true})
\item {\bfseries{delimiter}} The default delimiter for segments, e.\+g. {\ttfamily \mbox{[}\texorpdfstring{$^\wedge$}{\string^}/\#?\mbox{]}} for {\ttfamily \+:named} patterns. (default\+: `'/\#?\textquotesingle{}{\ttfamily )}

\item {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}encode\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} A function to encode strings before inserting into}Reg\+Exp{\ttfamily . (default\+:}x =\texorpdfstring{$>$}{>} x{\ttfamily )}
\item {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}prefixes\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} List of characters to automatically consider prefixes when parsing. (default\+:}./\`{})
