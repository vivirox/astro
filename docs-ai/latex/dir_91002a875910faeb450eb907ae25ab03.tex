This package is a utility that turns a string of Java\+Script into an esast (estree with some extra cleanliness) syntax tree.\hypertarget{readme.md_autotoc_md12751}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md12751}
You can use this utility when you want to deal with ASTs of Java\+Script combined with other \href{https://github.com/syntax-tree/unist}{\texttt{ unist}} and \href{https://github.com/vfile/vfile}{\texttt{ {\ttfamily vfile}}} things. You can use \href{https://github.com/acornjs/acorn}{\texttt{ {\ttfamily acorn}}} itself if you don’t care about unified.

The utility \href{https://github.com/syntax-tree/estree-util-to-js}{\texttt{ {\ttfamily estree-\/util-\/to-\/js}}} does the inverse of this utility. It turns the tree into a string of Java\+Script.\hypertarget{readme.md_autotoc_md12752}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md12752}
