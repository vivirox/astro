\end{DoxyItemize}\hypertarget{README.md_autotoc_md28531}{}\doxysubsection{\texorpdfstring{Functions}{Functions}}\label{README.md_autotoc_md28531}
For all of the following functions, argument {\ttfamily expression} is the target string, and {\ttfamily needle} is the regex pattern to search for.


\begin{DoxyItemize}
\item Argument {\ttfamily expression} (the string being searched through) is assumed to be a flag-\/{\ttfamily v}-\/mode regex pattern string. In other words, nested character classes within it are supported when determining the context for a match.
\item Argument {\ttfamily needle} (the regex pattern being searched for) is provided as a string, and is applied with flags {\ttfamily su}.
\item If argument {\ttfamily context} is not provided, matches are allowed in all contexts. In other words, inside and outside of character classes.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md28532}{}\doxysubsubsection{\texorpdfstring{{\ttfamily replace\+Unescaped}}{{\ttfamily replace\+Unescaped}}}\label{README.md_autotoc_md28532}

Extracts the full contents of a group (subpattern) from the given expression, accounting for escaped characters, nested groups, and character classes. The group is identified by the position where its contents start (the string index just after the group\textquotesingle{}s opening delimiter). Returns the rest of the string if the group is unclosed.
