\begin{quote}
Utility module to print pretty messages on SIGINFO/\+SIGUSR1 \end{quote}


{\ttfamily SIGINFO} on BSD / mac\+OS and {\ttfamily SIGUSR1} on Linux, usually triggered by {\ttfamily Ctrl + T}, are by convention used to print information about a long running process internal state. Eg. {\ttfamily dd} will tell you how many blocks it has written and at what speed, while {\ttfamily xz} will tell you progress, compression ratio and estimated time remaining.

This module wraps both signals, checks if the process is connected to TTY and lets you do whatever you want.\hypertarget{README.md_autotoc_md30716}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md30716}
