Recharts is a {\bfseries{Redefined}} chart library built with \href{https://facebook.github.io/react/}{\texttt{ React}} and \href{http://d3js.org}{\texttt{ D3}}.

The main purpose of this library is to help you to write charts in React applications without any pain. Main principles of Recharts are\+:


\begin{DoxyEnumerate}
\item {\bfseries{Simply}} deploy with React components.
\item {\bfseries{Native}} SVG support, lightweight depending only on some D3 submodules.
\item {\bfseries{Declarative}} components, components of charts are purely presentational.
\end{DoxyEnumerate}

Documentation at \href{https://recharts.org}{\texttt{ recharts.\+org}} and our \href{https://release--63da8268a0da9970db6992aa.chromatic.com/}{\texttt{ storybook (WIP)}}

Please see \href{https://github.com/recharts/recharts/wiki}{\texttt{ the wiki}} for FAQ.

All development is done on the {\ttfamily master} branch. The current latest release and storybook documentation reflects what is on the {\ttfamily release} branch.\hypertarget{README.md_autotoc_md27924}{}\doxysubsection{\texorpdfstring{Examples}{Examples}}\label{README.md_autotoc_md27924}
