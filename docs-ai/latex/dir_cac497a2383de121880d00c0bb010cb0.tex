\doxysubsection{Detailed Description}
Language services for CSS, LESS and SCSS

\href{https://www.npmjs.org/package/vscode-css-languageservice}{\texttt{ }} \href{https://npmjs.org/package/vscode-css-languageservice}{\texttt{ }} \href{https://github.com/microsoft/vscode-css-languageservice/actions}{\texttt{ }} \href{https://opensource.org/licenses/MIT}{\texttt{ }}\hypertarget{README.md_autotoc_md35278}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{README.md_autotoc_md35278}
The {\itshape vscode-\/css-\/languageservice} contains the language smarts behind the CSS, LESS and SCSS editing experience of Visual Studio Code and the Monaco editor.
\begin{DoxyItemize}
\item {\itshape do\+Validation} analyses an input string and returns syntax and lint errors.
\item {\itshape do\+Complete} provides completion proposals for a given location.
\item {\itshape do\+Hover} provides a hover text for a given location.
\item {\itshape find\+Definition} finds the definition of the symbol at the given location.
\item {\itshape find\+References} finds all references to the symbol at the given location.
\item {\itshape find\+Document\+Highlights} finds all symbols connected to the given location.
\item {\itshape find\+Document\+Symbols} provides all symbols in the given document
\item {\itshape do\+Code\+Actions} evaluates code actions for the given location, typically to fix a problem.
\item {\itshape find\+Document\+Colors} evaluates all color symbols in the given document
\item {\itshape do\+Rename} renames all symbols connected to the given location.
\item {\itshape prepare\+Rename} the range of the node that can be renamed
\item {\itshape get\+Folding\+Ranges} returns folding ranges in the given document.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md35279}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md35279}
