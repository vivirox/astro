An ES7/\+ES2016 spec-\/compliant {\ttfamily Array.\+prototype.\+includes} shim/polyfill/replacement that works as far down as ES3.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the proposed \href{https://262.ecma-international.org/6.0/}{\texttt{ spec}}.

Because {\ttfamily Array.\+prototype.\+includes} depends on a receiver (the {\ttfamily this} value), the main export takes the array to operate on as the first argument.
