\doxysubsection{Detailed Description}

{\ttfamily psl} is a {\ttfamily Java\+Script} domain name parser based on the \href{https://publicsuffix.org/}{\texttt{ Public Suffix List}}.

This implementation is tested against the \href{http://mxr.mozilla.org/mozilla-central/source/netwerk/test/unit/data/test_psl.txt?raw=1}{\texttt{ test data hosted by Mozilla}} and kindly provided by \href{https://www.comodo.com/}{\texttt{ Comodo}}.

Cross browser testing provided by \href{https://www.browserstack.com/}{\texttt{ }}\hypertarget{README.md_autotoc_md26685}{}\doxysubsection{\texorpdfstring{What is the Public Suffix List?}{What is the Public Suffix List?}}\label{README.md_autotoc_md26685}
The Public Suffix List is a cross-\/vendor initiative to provide an accurate list of domain name suffixes.

The Public Suffix List is an initiative of the Mozilla Project, but is maintained as a community resource. It is available for use in any software, but was originally created to meet the needs of browser manufacturers.

A "{}public suffix"{} is one under which Internet users can directly register names. Some examples of public suffixes are "{}.\+com"{}, "{}.\+co.\+uk"{} and "{}pvt.\+k12.\+wy.\+us"{}. The Public Suffix List is a list of all known public suffixes.

Source\+: \href{http://publicsuffix.org}{\texttt{ http\+://publicsuffix.\+org}}\hypertarget{README.md_autotoc_md26686}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md26686}

\hypertarget{README.md_autotoc_md26695}{}\doxysubsubsection{\texorpdfstring{{\ttfamily psl.\+parse(domain)}}{{\ttfamily psl.\+parse(domain)}}}\label{README.md_autotoc_md26695}
Parse domain based on Public Suffix List. Returns an {\ttfamily Object} with the following properties\+:


\begin{DoxyItemize}
\item {\ttfamily tld}\+: Top level domain (this is the {\itshape public suffix}).
\item {\ttfamily sld}\+: Second level domain (the first private part of the domain name).
\item {\ttfamily domain}\+: The domain name is the {\ttfamily sld} + {\ttfamily tld}.
\item {\ttfamily subdomain}\+: Optional parts left of the domain.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md26696}{}\doxysubsubsubsection{\texorpdfstring{Examples}{Examples}}\label{README.md_autotoc_md26696}
