\doxysubsection{Detailed Description}
\href{https://npmjs.com/package/tinyglobby}{\texttt{ }} \href{https://npmjs.com/package/tinyglobby}{\texttt{ }}

A fast and minimal alternative to globby and fast-\/glob, meant to behave the same way.

Both globby and fast-\/glob present some behavior no other globbing lib has, which makes it hard to manually replace with something smaller and better.

This library uses only two subdependencies, compared to {\ttfamily globby}\textquotesingle{}s \href{https://npmgraph.js.org/?q=globby@14.1.0}{\texttt{ 23}} and {\ttfamily fast-\/glob}\textquotesingle{}s \href{https://npmgraph.js.org/?q=fast-glob@3.3.3}{\texttt{ 17}}.\hypertarget{README.md_autotoc_md32863}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md32863}
