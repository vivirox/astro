\doxysubsection{Detailed Description}
A robust \& optimized polyfill for \href{http://ecma-international.org/ecma-262/6.0/\#sec-string.prototype.repeat}{\texttt{ the {\ttfamily String.\+prototype.\+repeat} method in ECMAScript 6}}.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{https://tc39.es/ecma262/\#sec-string.prototype.repeat}{\texttt{ spec}}.
