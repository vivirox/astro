\doxysubsection{Detailed Description}
\href{https://www.npmjs.com/package/eventsource-parser}{\texttt{ }}\href{https://bundlephobia.com/result?p=eventsource-parser}{\texttt{ }}

A streaming parser for \href{https://developer.mozilla.org/en-US/docs/Web/API/Server-sent_events}{\texttt{ server-\/sent events/eventsource}}, without any assumptions about how the actual stream of data is retrieved. It is intended to be a building block for clients and polyfills in javascript environments such as browsers, node.\+js and deno.

You create an instance of the parser, and {\itshape feed} it chunks of data -\/ partial or complete, and the parse emits parsed messages once it receives a complete message.\hypertarget{README.md_autotoc_md14753}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md14753}
