\doxysubsection{Detailed Description}
Turns off all rules that are unnecessary or might conflict with \href{https://github.com/prettier/prettier}{\texttt{ Prettier}}.

This lets you use your favorite shareable config without letting its stylistic choices get in the way when using Prettier.

Note that this config {\itshape only} turns rules {\itshape off,} so it only makes sense using it together with some other config.
