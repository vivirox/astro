{\ttfamily Although it should generally not be necessary because all escapes are handled by default, escape decoding can be customized using the {\ttfamily get\+Unescaped} option.}

{\ttfamily The following is the default, equivalent to not setting the {\ttfamily get\+Unescaped} option.}

{\ttfamily 
\begin{DoxyCode}{0}

\end{DoxyCode}
}

{\ttfamily Included {\ttfamily get\+Unescaped} implementations\+:}

{\ttfamily 
\begin{DoxyItemize}
\item {\ttfamily get\+Unescaped\+Any} -\/ Decode {\itshape ANY} Javascript supported escape sequence.
\end{DoxyItemize}}

{\ttfamily A custom {\ttfamily get\+Unescaped} implementation receives the escape sequence as the first argument, and the escape sequence code point number or {\ttfamily null} (for single letter escape sequences) as the second argument. It can return {\ttfamily true} to use the standard decoding, {\ttfamily false} to treat the sequence as invalid (only removes the leading backslash), or a string (non-\/zero length) to provide a custom decoded value for the escape sequence.}
