\doxysubsection{Detailed Description}
\begin{quote}
Strip leading whitespace from each line in a string \end{quote}


The line with the least number of leading whitespace, ignoring empty lines, determines the number to remove.

Useful for removing redundant indentation.\hypertarget{readme.md_autotoc_md28018}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md28018}
