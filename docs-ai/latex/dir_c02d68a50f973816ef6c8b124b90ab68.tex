This package detects whether text is written left-\/to-\/right or right-\/to-\/left.\hypertarget{readme.md_autotoc_md11993}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md11993}
This is a simple and fast algorithm. It looks at the first strong left-\/to-\/right or right-\/to-\/left character (for example, the letter {\ttfamily a} is LTR, the letter {\ttfamily ى} is RTL).
