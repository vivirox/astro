{\bfseries{vfile}} is a small and browser friendly virtual file format that tracks metadata (such as a fileâ€™s {\ttfamily path} and {\ttfamily contents}) and messages.

It was made specifically for {\bfseries{\href{https://github.com/unifiedjs/unified}{\texttt{ unified}}}} and generally for the common task of parsing, transforming, and serializing data, where {\ttfamily vfile} handles everything about the document being compiled. This is useful for example when building linters, compilers, static site generators, or other build tools. {\bfseries{vfile}} is part of the \href{https://unifiedjs.com}{\texttt{ unified collective}}.
