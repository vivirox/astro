\doxysubsection{Detailed Description}
Module for verifying whether an object has been garbage collected or not.

Internally creates a weak reference to the object, and forces garbage collection to happen. If the reference is gone, it meant no one else was pointing to the object.\hypertarget{README.md_autotoc_md20044}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{README.md_autotoc_md20044}
