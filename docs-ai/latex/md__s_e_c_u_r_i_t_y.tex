If you discover a security vulnerability within this project, please send an email to \mbox{[}SECURITY\+\_\+\+EMAIL\mbox{]}. All security vulnerabilities will be promptly addressed.

Please include the following information in your report\+:
\begin{DoxyItemize}
\item Type of vulnerability
\item Full path of source file(s) related to the vulnerability
\item Location of the affected source code (tag/branch/commit or direct URL)
\item Step-\/by-\/step instructions to reproduce the vulnerability
\item Proof-\/of-\/concept or exploit code (if possible)
\item Impact of the vulnerability

\hypertarget{md__s_e_c_u_r_i_t_y_autotoc_md36768}{}\doxysubsection{\texorpdfstring{1. Access Control}{1. Access Control}}\label{md__s_e_c_u_r_i_t_y_autotoc_md36768}

\begin{DoxyItemize}
\item All repository access must use 2FA (Two-\/\+Factor Authentication)
\item SSH keys must be rotated every 90 days
\item Personal Access Tokens (PATs) must\+:
\begin{DoxyItemize}
\item Have limited scope
\item Expire after 30 days
\item Never be shared or committed to the repository
\end{DoxyItemize}
\item Branch protection rules enforce\+:
\begin{DoxyItemize}
\item Required reviews from at least 2 team members
\item Required status checks to pass before merging
\item No direct pushes to main/master branch
\item Linear history (no merge commits)
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md__s_e_c_u_r_i_t_y_autotoc_md36769}{}\doxysubsection{\texorpdfstring{2. Code Security}{2. Code Security}}\label{md__s_e_c_u_r_i_t_y_autotoc_md36769}

\begin{DoxyItemize}
\item All code must pass automated security scanning before merge, including\+:
\begin{DoxyItemize}
\item Static Application Security Testing (SAST)
\item Software Composition Analysis (SCA)
\item Secret scanning
\item Container scanning
\item Infrastructure as Code (IaC) scanning
\item Dependency vulnerability scanning
\end{DoxyItemize}
\item Automated scanning configuration\+:
\begin{DoxyItemize}
\item SAST scanning on every pull request
\item Weekly full repository scans
\item Daily dependency vulnerability scans
\item Real-\/time secret detection
\item Malware detection in dependencies
\end{DoxyItemize}
\item Scanning requirements\+:
\begin{DoxyItemize}
\item Zero high-\/severity issues allowed in main branch
\item All critical vulnerabilities must be addressed within 24 hours
\item Medium severity issues must be addressed within 7 days
\item Dependencies must be kept up-\/to-\/date with no known vulnerabilities
\item All security warnings must be resolved or have documented exceptions
\end{DoxyItemize}
\item Code signing required for all commits using GPG keys
\item Automated security tools configured\+:
\begin{DoxyItemize}
\item Code\+QL for SAST
\item Dependabot for dependency scanning
\item Git\+Hub Advanced Security features enabled
\item Container scanning using Trivy
\item IaC scanning using Checkov
\end{DoxyItemize}
\item Security scan reports\+:
\begin{DoxyItemize}
\item Automated reports generated after each scan
\item Weekly security metrics collection
\item Monthly trend analysis
\item Quarterly security posture review
\end{DoxyItemize}
