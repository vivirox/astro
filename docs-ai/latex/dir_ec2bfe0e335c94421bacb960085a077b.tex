Assertion Error is a module that contains two classes\+: {\ttfamily Assertion\+Error}, which is an instance of an {\ttfamily Error}, and {\ttfamily Assertion\+Result} which is not an instance of Error.

These can be useful for returning from a function -\/ if the function "{}succeeds"{} return an {\ttfamily Assertion\+Result} and if the function fails return (or throw) an {\ttfamily Assertion\+Error}.

Both {\ttfamily Assertion\+Error} and {\ttfamily Assertion\+Result} implement the {\ttfamily Result} interface\+:
