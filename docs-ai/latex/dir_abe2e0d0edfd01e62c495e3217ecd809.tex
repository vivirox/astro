\hypertarget{README.md_autotoc_md11305}{}\doxysubsubsection{\texorpdfstring{data\+Uri\+To\+Buffer(\+String uri) → Buffer}{data\+Uri\+To\+Buffer(\+String uri) → Buffer}}\label{README.md_autotoc_md11305}
The {\ttfamily type} property on the Buffer instance gets set to the main type portion of the "{}mediatype"{} portion of the "{}data"{} URI, or defaults to {\ttfamily "{}text/plain"{}} if not specified.

The {\ttfamily type\+Full} property on the Buffer instance gets set to the entire "{}mediatype"{} portion of the "{}data"{} URI (including all parameters), or defaults to {\ttfamily "{}text/plain;charset=\+US-\/\+ASCII"{}} if not specified.

The {\ttfamily charset} property on the Buffer instance gets set to the Charset portion of the "{}mediatype"{} portion of the "{}data"{} URI, or defaults to {\ttfamily "{}\+US-\/\+ASCII"{}} if the entire type is not specified, or defaults to {\ttfamily "{}"{}} otherwise.

{\itshape Note}\+: If the only the main type is specified but not the charset, e.\+g. {\ttfamily "{}data\+:text/plain,abc"{}}, the charset is set to the empty string.
