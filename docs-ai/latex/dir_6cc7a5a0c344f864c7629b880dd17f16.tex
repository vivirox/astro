\doxysubsection{Detailed Description}
Parse e-\/mail address fields. Input can be a single address ({\ttfamily "{}andris@kreata.\+ee"{}}), a formatted address ({\ttfamily "{}\+Andris Reinman $<$andris@kreata.\+ee$>$"{}}), comma separated list of addresses ({\ttfamily "{}andris@kreata.\+ee, andris.\+reinman@kreata.\+ee"{}}), an address group ({\ttfamily "{}disclosed-\/recipients\+:andris@kreata.\+ee;"{}}) or a mix of all the formats.

In addition to comma the semicolon is treated as the list delimiter as well (except when used in the group syntax), so a value {\ttfamily "{}andris@kreata.\+ee; andris.\+reinman@kreata.\+ee"{}} is identical to {\ttfamily "{}andris@kreata.\+ee, andris.\+reinman@kreata.\+ee"{}}.\hypertarget{README.md_autotoc_md7670}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md7670}
