\hypertarget{README.md_autotoc_md11685}{}\doxysubsection{\texorpdfstring{Summary}{Summary}}\label{README.md_autotoc_md11685}
this module exports a function that takes 2 arguments\+: {\ttfamily options} and {\ttfamily defaults}. When called, it overrides all of {\ttfamily undefined} properties in {\ttfamily options} with the clones of properties defined in {\ttfamily defaults}

Sidecases\+: if called with a falsy {\ttfamily options} value, options will be initialized to a new object before being merged onto.
