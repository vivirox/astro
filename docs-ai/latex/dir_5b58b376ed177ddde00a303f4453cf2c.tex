This is a package that can parse BCP 47 language tags to an object representing them, and serialize those objects back into language tags. It supports a forgiving mode to handle incorrect BCP 47 tags and can emit warnings about problems in incorrect tags.\hypertarget{readme.md_autotoc_md9179}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md9179}
You can use this package if you need to access the data stored in BCP 47 language tags. You can also use this package if you want to check (lint) or manipulate tags.\hypertarget{readme.md_autotoc_md9180}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md9180}

Parse a BCP 47 tag into a language schema. Note that the algorithm is case insensitive.\hypertarget{readme.md_autotoc_md9184}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily options.\+normalize}}{{\ttfamily options.\+normalize}}}\label{readme.md_autotoc_md9184}
Whether to normalize legacy tags when possible ({\ttfamily boolean}, default\+: {\ttfamily true}). For example, {\ttfamily i-\/klingon} does not match the BCP 47 language algorithm but is considered valid by BCP 47 nonetheless. It is suggested to use {\ttfamily tlh} instead (the ISO 639-\/3 code for Klingon). When {\ttfamily normalize} is {\ttfamily true}, passing {\ttfamily i-\/klingon} or other deprecated tags, is handled as if their suggested valid tag was given instead.\hypertarget{readme.md_autotoc_md9185}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily options.\+forgiving}}{{\ttfamily options.\+forgiving}}}\label{readme.md_autotoc_md9185}
By default, when an error is encountered, an empty object is returned. When in forgiving mode, all found values up to the point of the error are included ({\ttfamily boolean}, default\+: {\ttfamily false}). So, for example, where by default {\ttfamily en-\/\+GB-\/abcdefghi} an empty object is returned (as the language variant is too long), in {\ttfamily forgiving} mode the {\ttfamily language} of {\ttfamily schema} is populated with {\ttfamily en} and the {\ttfamily region} is populated with {\ttfamily GB}.\hypertarget{readme.md_autotoc_md9186}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily options.\+warning}}{{\ttfamily options.\+warning}}}\label{readme.md_autotoc_md9186}
When given, {\ttfamily warning} is called when an error is encountered (\`{}\+Function\`{}).\hypertarget{readme.md_autotoc_md9187}{}\doxysubsubsubsubsubsection{\texorpdfstring{Returns}{Returns}}\label{readme.md_autotoc_md9187}
Parsed BCP 47 language tag (\`{}\+Schema\`{}).\hypertarget{readme.md_autotoc_md9188}{}\doxysubsubsubsubsubsection{\texorpdfstring{Throws}{Throws}}\label{readme.md_autotoc_md9188}

BCP 47 language tag ({\ttfamily string}).\hypertarget{readme.md_autotoc_md9191}{}\doxysubsubsection{\texorpdfstring{{\ttfamily Schema}}{{\ttfamily Schema}}}\label{readme.md_autotoc_md9191}
A schema represents a language tag. A schema is deemed empty when it has neither {\ttfamily language}, {\ttfamily irregular}, {\ttfamily regular}, nor {\ttfamily privateuse} (where an empty {\ttfamily privateuse} array is handled as no {\ttfamily privateuse} as well).\hypertarget{readme.md_autotoc_md9192}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+language}}{{\ttfamily schema.\+language}}}\label{readme.md_autotoc_md9192}
Two or three character \href{https://en.wikipedia.org/wiki/ISO_639}{\texttt{ ISO 639}} language code, four character reserved language code, or 5 to 8 (inclusive) characters registered language subtag ({\ttfamily string}). For example, {\ttfamily en} (English) or {\ttfamily cmn} (Mandarin Chinese).\hypertarget{readme.md_autotoc_md9193}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+extended\+Language\+Subtags}}{{\ttfamily schema.\+extended\+Language\+Subtags}}}\label{readme.md_autotoc_md9193}
Selected three-\/character \href{https://en.wikipedia.org/wiki/ISO_639}{\texttt{ ISO 639}} codes({\ttfamily Array\texorpdfstring{$<$}{<}string\texorpdfstring{$>$}{>}}), such as {\ttfamily yue} in {\ttfamily zh-\/yue-\/\+HK} (Chinese, Cantonese, as used in Hong Kong SAR).\hypertarget{readme.md_autotoc_md9194}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+script}}{{\ttfamily schema.\+script}}}\label{readme.md_autotoc_md9194}
Four character \href{https://en.wikipedia.org/wiki/ISO_15924}{\texttt{ ISO 15924}} script code ({\ttfamily string}), such as {\ttfamily Latn} in {\ttfamily hy-\/\+Latn-\/\+IT-\/arevela} (Eastern Armenian written in Latin script, as used in Italy).\hypertarget{readme.md_autotoc_md9195}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+region}}{{\ttfamily schema.\+region}}}\label{readme.md_autotoc_md9195}
Two alphabetical character \href{https://en.wikipedia.org/wiki/ISO_3166-1}{\texttt{ ISO 3166-\/1}} code or three digit \href{https://en.wikipedia.org/wiki/UN_M.49}{\texttt{ UN M49}} code ({\ttfamily string}). For example, {\ttfamily CN} in {\ttfamily cmn-\/\+Hans-\/\+CN} (Mandarin Chinese, Simplified script, as used in China) or {\ttfamily 419} in {\ttfamily es-\/419} (Spanish as used in Latin America and the Caribbean).\hypertarget{readme.md_autotoc_md9196}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+variants}}{{\ttfamily schema.\+variants}}}\label{readme.md_autotoc_md9196}
5 to 8 (inclusive) character language variants ({\ttfamily Array\texorpdfstring{$<$}{<}string\texorpdfstring{$>$}{>}}), such as {\ttfamily rozaj} and {\ttfamily biske} in {\ttfamily sl-\/rozaj-\/biske} (San Giorgio dialect of Resian dialect of Slovenian).\hypertarget{readme.md_autotoc_md9197}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+extensions}}{{\ttfamily schema.\+extensions}}}\label{readme.md_autotoc_md9197}
List of extensions ({\ttfamily Array\texorpdfstring{$<$}{<}Object\texorpdfstring{$>$}{>}}), each an object containing a one character {\ttfamily singleton}, and a list of {\ttfamily extensions} ({\ttfamily string}). {\ttfamily singleton} cannot be {\ttfamily x} (case insensitive) and {\ttfamily extensions} must be between two and eight (inclusive) characters. For example, an extension would be {\ttfamily u-\/co-\/phonebk} in {\ttfamily de-\/\+DE-\/u-\/co-\/phonebk} (German, as used in Germany, using German phonebook sort order), where {\ttfamily u} is the {\ttfamily singleton} and {\ttfamily co} and {\ttfamily phonebk} are its extensions.\hypertarget{readme.md_autotoc_md9198}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+privateuse}}{{\ttfamily schema.\+privateuse}}}\label{readme.md_autotoc_md9198}
List of private-\/use subtags ({\ttfamily Array\texorpdfstring{$<$}{<}string\texorpdfstring{$>$}{>}}), where each subtag must be between one and eight (inclusive) characters.\hypertarget{readme.md_autotoc_md9199}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily schema.\+regular}}{{\ttfamily schema.\+regular}}}\label{readme.md_autotoc_md9199}
