Some APIs allow you to make conditional requests by providing an {\ttfamily If-\/\+None-\/\+Match} or {\ttfamily If-\/\+Modified-\/\+Since} header. This can be used to avoid downloading the same data multiple times. These helpers allow you to store the etag or last-\/modified value from a response and then use it to make a conditional request when updating later.


\begin{DoxyItemize}
\item {\ttfamily get\+Conditional\+Headers}
\end{DoxyItemize}

Get the headers needed to make a conditional request. Uses the etag and last-\/modified values from the meta store, and sets the {\ttfamily If-\/\+None-\/\+Match} or {\ttfamily If-\/\+Modified-\/\+Since} headers.


\begin{DoxyItemize}
\item {\ttfamily store\+Conditional\+Headers}
\end{DoxyItemize}

Store the {\ttfamily ETag} or {\ttfamily Last-\/\+Modified} headers from a response in the meta store.
