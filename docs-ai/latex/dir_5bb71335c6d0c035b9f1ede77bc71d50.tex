The Vercel AI SDK is {\bfseries{a library for building AI-\/powered streaming text and chat UIs}}.\hypertarget{README.md_autotoc_md7680}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md7680}

\begin{DoxyItemize}
\item \href{https://swr.vercel.app}{\texttt{ SWR}}-\/powered React, Svelte, Vue and Solid helpers for streaming text responses and building chat and completion UIs
\item First-\/class support for \href{https://js.langchain.com/docs}{\texttt{ Lang\+Chain}} and \href{https://openai.com}{\texttt{ Open\+AI}}, \href{https://www.anthropic.com}{\texttt{ Anthropic}}, \href{https://cohere.com}{\texttt{ Cohere}}, \href{https://huggingface.co}{\texttt{ Hugging Face}}, \href{https://app.fireworks.ai}{\texttt{ Fireworks}} and \href{https://replicate.com}{\texttt{ Replicate}}
\item Node.\+js, Serverless, and \href{https://edge-runtime.vercel.app/}{\texttt{ Edge Runtime}} support
\item Callbacks for saving completed streaming responses to a database (in the same request)
