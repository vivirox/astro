\hypertarget{Readme.md_autotoc_md26040}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{Readme.md_autotoc_md26040}
This module uses a pre-\/defined list of rules, applied in order, to singularize or pluralize a given word. There are many cases where this is useful, such as any automation based on user input. For applications where the word(s) are known ahead of time, you can use a simple ternary (or function) which would be a much lighter alternative.\hypertarget{Readme.md_autotoc_md26041}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{Readme.md_autotoc_md26041}

\begin{DoxyItemize}
\item {\ttfamily word\+: string} The word to pluralize
\item {\ttfamily count\+: number} How many of the word exist
\item {\ttfamily inclusive\+: boolean} Whether to prefix with the number (e.\+g. 3 ducks)
\end{DoxyItemize}
