\doxysubsection{Detailed Description}
{\ttfamily base64-\/js} does basic base64 encoding/decoding in pure JS.

\href{http://travis-ci.org/beatgammit/base64-js}{\texttt{ }}

\href{https://ci.testling.com/beatgammit/base64-js}{\texttt{ }}

Many browsers already have base64 encoding/decoding functionality, but it is for text data, not all-\/purpose binary data.

Sometimes encoding/decoding binary data in the browser is useful, and that is what this module does.\hypertarget{README.md_autotoc_md21600}{}\doxysubsection{\texorpdfstring{install}{install}}\label{README.md_autotoc_md21600}
