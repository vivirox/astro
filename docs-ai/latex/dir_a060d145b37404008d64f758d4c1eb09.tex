Handle one value. Based on the bound {\ttfamily key}, a respective handler will be called. If {\ttfamily value} is not an object, or doesn’t have a {\ttfamily key} property, the special “invalid” handler will be called. If {\ttfamily value} has an unknown {\ttfamily key}, the special “unknown” handler will be called.

All arguments, and the context object ({\ttfamily this}), are passed through to the handler, and it’s result is returned.\hypertarget{readme.md_autotoc_md36736}{}\doxysubsubsubsubsubsection{\texorpdfstring{{\ttfamily one.\+handlers}}{{\ttfamily one.\+handlers}}}\label{readme.md_autotoc_md36736}
