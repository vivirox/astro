\doxysubsection{Detailed Description}
A robust \& optimized polyfill for \href{http://people.mozilla.org/~jorendorff/es6-draft.html\#sec-string.prototype.includes}{\texttt{ the {\ttfamily String.\+prototype.\+includes} method (previously known as {\ttfamily String.\+prototype.\+contains}) in ECMAScript 6}}.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{https://tc39.es/ecma262/\#sec-string.prototype.includes}{\texttt{ spec}}.
