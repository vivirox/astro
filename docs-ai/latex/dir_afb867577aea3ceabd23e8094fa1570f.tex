\doxysubsection{Detailed Description}
Draco is an open-\/source library for compressing and decompressing 3D geometric meshes and point clouds. It is intended to improve the storage and transmission of 3D graphics.

\href{https://google.github.io/draco/}{\texttt{ Website}} \texorpdfstring{$\vert$}{|} \href{https://github.com/google/draco}{\texttt{ Git\+Hub}}\hypertarget{README.md_autotoc_md32807}{}\doxysubsection{\texorpdfstring{Contents}{Contents}}\label{README.md_autotoc_md32807}
This folder contains three utilities\+:


\begin{DoxyItemize}
\item {\ttfamily draco\+\_\+decoder.\+js} — Emscripten-\/compiled decoder, compatible with any modern browser.
\item {\ttfamily draco\+\_\+decoder.\+wasm} — Web\+Assembly decoder, compatible with newer browsers and devices.
\item {\ttfamily draco\+\_\+wasm\+\_\+wrapper.\+js} — Java\+Script wrapper for the WASM decoder.
\end{DoxyItemize}

Each file is provided in two variations\+:


\begin{DoxyItemize}
\item {\bfseries{Default\+:}} Latest stable builds, tracking the project\textquotesingle{}s \href{https://github.com/google/draco}{\texttt{ master branch}}.
\item {\bfseries{gl\+TF\+:}} Builds targeted by the \href{https://github.com/KhronosGroup/glTF/tree/master/extensions/2.0/Khronos/KHR_draco_mesh_compression}{\texttt{ gl\+TF mesh compression extension}}, tracking the \href{https://github.com/google/draco/tree/gltf_2.0_draco_extension}{\texttt{ corresponding Draco branch}}.
\end{DoxyItemize}

Either variation may be used with {\ttfamily THREE.\+DRACOLoader}\+:
