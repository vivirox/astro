This package is a tiny utility that gets the textual content of a node.\hypertarget{readme.md_autotoc_md22832}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md22832}
This utility is useful when you have a node, say a heading, and want to get the text inside it.

This package does not serialize markdown, that’s what \href{https://github.com/syntax-tree/mdast-util-to-markdown}{\texttt{ {\ttfamily mdast-\/util-\/to-\/markdown}}} does.

Similar packages, \href{https://github.com/rehypejs/rehype-minify/tree/main/packages/hast-util-to-string}{\texttt{ {\ttfamily hast-\/util-\/to-\/string}}} and \href{https://github.com/syntax-tree/hast-util-to-text}{\texttt{ {\ttfamily hast-\/util-\/to-\/text}}}, do the same but on \href{https://github.com/syntax-tree/hast}{\texttt{ hast}}.\hypertarget{readme.md_autotoc_md22833}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22833}
