This package contains extensions that add support for all features enabled by GFM to \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}}. It supports autolink literals, footnotes, strikethrough, tables, tagfilter, and tasklists.\hypertarget{readme.md_autotoc_md23303}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md23303}
This project is useful when you want to support GFM in markdown.

You can use these extensions when you are working with \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}}. Alternatively, you can also use the underlying features separately\+:


\begin{DoxyItemize}
\item \href{https://github.com/micromark/micromark-extension-gfm-autolink-literal}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/autolink-\/literal}}} — support GFM \href{https://github.github.com/gfm/\#autolinks-extension-}{\texttt{ autolink literals}}
\item \href{https://github.com/micromark/micromark-extension-gfm-footnote}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/footnote}}} — support GFM \href{https://github.blog/changelog/2021-09-30-footnotes-now-supported-in-markdown-fields/}{\texttt{ footnotes}}
\item \href{https://github.com/micromark/micromark-extension-gfm-strikethrough}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/strikethrough}}} — support GFM \href{https://github.github.com/gfm/\#strikethrough-extension-}{\texttt{ strikethrough}}
\item \href{https://github.com/micromark/micromark-extension-gfm-table}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/table}}} — support GFM \href{https://github.github.com/gfm/\#tables-extension-}{\texttt{ tables}}
\item \href{https://github.com/micromark/micromark-extension-gfm-tagfilter}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/tagfilter}}} — support GFM \href{https://github.github.com/gfm/\#disallowed-raw-html-extension-}{\texttt{ tagfilter}}
\item \href{https://github.com/micromark/micromark-extension-gfm-task-list-item}{\texttt{ {\ttfamily micromark-\/extension-\/gfm-\/task-\/list-\/item}}} — support GFM \href{https://github.github.com/gfm/\#task-list-items-extension-}{\texttt{ tasklists}}
\end{DoxyItemize}

When you need a syntax tree, combine this package with \href{https://github.com/syntax-tree/mdast-util-gfm}{\texttt{ {\ttfamily mdast-\/util-\/gfm}}}.

All these packages are used in \href{https://github.com/remarkjs/remark-gfm}{\texttt{ {\ttfamily remark-\/gfm}}}, which focusses on making it easier to transform content by abstracting these internals away.\hypertarget{readme.md_autotoc_md23304}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md23304}
