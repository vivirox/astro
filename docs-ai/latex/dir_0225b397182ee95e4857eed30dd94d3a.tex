\doxysubsection{Detailed Description}
{\itshape emoji-\/regex} offers a regular expression to match all emoji symbols and sequences (including textual representations of emoji) as per the Unicode Standard.

This repository contains a script that generates this regular expression based on \href{https://github.com/node-unicode/node-unicode-data}{\texttt{ Unicode data}}. Because of this, the regular expression can easily be updated whenever new emoji are added to the Unicode standard.\hypertarget{README.md_autotoc_md13985}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md13985}
