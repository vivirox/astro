This is a map from the \href{https://html.spec.whatwg.org/multipage/parsing.html\#table-charref-overrides}{\texttt{ HTML spec}} of C1 ASCII/\+Unicode control characters (which are disallowed by HTML) to the characters those code points would have in Windows 1252. For example, U+0080 (Padding Character) maps to {\ttfamily €}, because that’s used for 0x80 in Windows 1252.\hypertarget{readme.md_autotoc_md10080}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md10080}
Probably never, unless you’re dealing with parsing HTML or similar XML-\/like things, or in a place where Unicode is not the primary encoding (it is in most places).\hypertarget{readme.md_autotoc_md10081}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md10081}
