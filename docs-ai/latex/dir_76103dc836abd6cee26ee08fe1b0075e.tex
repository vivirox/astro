\doxysubsection{Detailed Description}

\begin{DoxyItemize}
\item Fast and low memory CSS selector parser.
\item Parses CSS selector into object-\/model (AST).
\item Compliant with all historical and modern CSS specs.
\item Covered with tests.
\item Documented.
\item Supported CSS selector standards\+:
\begin{DoxyItemize}
\item {\ttfamily css1}\+: \href{https://www.w3.org/TR/CSS1/}{\texttt{ https\+://www.\+w3.\+org/\+TR/\+CSS1/}}
\item {\ttfamily css2}\+: \href{https://www.w3.org/TR/CSS2/}{\texttt{ https\+://www.\+w3.\+org/\+TR/\+CSS2/}}
\item {\ttfamily css3}/{\ttfamily selectors-\/3}\+: \href{https://www.w3.org/TR/selectors-3/}{\texttt{ https\+://www.\+w3.\+org/\+TR/selectors-\/3/}}
\item {\ttfamily selectors-\/4}\+: \href{https://www.w3.org/TR/selectors-4/}{\texttt{ https\+://www.\+w3.\+org/\+TR/selectors-\/4/}}
\item {\ttfamily latest}\+: refers to {\ttfamily selectors-\/4}
\item {\ttfamily progressive}\+: {\ttfamily latest} + accepts unknown psudo-\/classes, psudo-\/elements and attribute case sensitivity modifiers
\end{DoxyItemize}
\end{DoxyItemize}

Supported CSS modules\+:


\begin{DoxyItemize}
\item {\ttfamily css-\/position-\/1}, {\ttfamily css-\/position-\/2}, {\ttfamily css-\/position-\/3}, {\ttfamily css-\/position-\/4}\+: Adds position-\/related pseudo-\/classes
\item {\ttfamily css-\/scoping-\/1}\+: Adds Shadow DOM selectors like {\ttfamily \+:host}, {\ttfamily \+:host-\/context()}, and {\ttfamily \+::slotted()}
\item {\ttfamily css-\/pseudo-\/4}\+: Adds modern pseudo-\/elements like {\ttfamily \+::selection}, {\ttfamily \+::backdrop}, etc.
\item {\ttfamily css-\/shadow-\/parts-\/1}\+: Adds {\ttfamily \+::part()} for styling shadow DOM components
\end{DoxyItemize}
