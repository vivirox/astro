This package contains an extension that adds support for the syntax enabled by \href{https://mdxjs.com}{\texttt{ MDX}} to \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}}. This extension is used inside MDX. It supports export/imports, expressions, JSX, and turns some markdown features off.\hypertarget{readme.md_autotoc_md23455}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md23455}
This project is useful when you want to support MDX.

You can use this extension when you are working with \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}}.

This package is aware of Java\+Script syntax. If you want to be unaware of Java\+Script, for example because expressions can include Rust or variables or whatnot, use \href{https://github.com/micromark/micromark-extension-mdx}{\texttt{ {\ttfamily micromark-\/extension-\/mdx}}}.

Alternatively, you can also use the underlying syntax extensions separately\+:


\begin{DoxyItemize}
\item \href{https://github.com/micromark/micromark-extension-mdx-expression}{\texttt{ {\ttfamily micromark-\/extension-\/mdx-\/expression}}} — support MDX expressions
\item \href{https://github.com/micromark/micromark-extension-mdx-jsx}{\texttt{ {\ttfamily micromark-\/extension-\/mdx-\/jsx}}} — support MDX JSX
\item \href{https://github.com/micromark/micromark-extension-mdx-md}{\texttt{ {\ttfamily micromark-\/extension-\/mdx-\/md}}} — turn some Common\+Mark features off
\item \href{https://github.com/micromark/micromark-extension-mdxjs-esm}{\texttt{ {\ttfamily micromark-\/extension-\/mdxjs-\/esm}}} — support MDX export/imports
\end{DoxyItemize}

When you need a syntax tree, combine this package with \href{https://github.com/syntax-tree/mdast-util-mdx}{\texttt{ {\ttfamily mdast-\/util-\/mdx}}}.

All these packages are used in \href{https://mdxjs.com/packages/remark-mdx/}{\texttt{ {\ttfamily remark-\/mdx}}}, which focusses on making it easier to transform content by abstracting these internals away.\hypertarget{readme.md_autotoc_md23456}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md23456}
