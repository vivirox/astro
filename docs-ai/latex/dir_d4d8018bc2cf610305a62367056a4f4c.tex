\doxysubsection{Detailed Description}
This extension adds to Ka\+TeX the {\ttfamily \textbackslash{}ce} and {\ttfamily \textbackslash{}pu} functions from the \href{https://mhchem.github.io/MathJax-mhchem/}{\texttt{ mhchem}} package.\hypertarget{README.md_autotoc_md21337}{}\doxysubsubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md21337}
This extension isn\textquotesingle{}t part of core Ka\+TeX, so the script should be separately included. Write the following line into the HTML page\textquotesingle{}s {\ttfamily \texorpdfstring{$<$}{<}head\texorpdfstring{$>$}{>}}. Place it {\itshape after} the line that calls {\ttfamily katex.\+js}, and if you make use of the \href{https://katex.org/docs/autorender.html}{\texttt{ auto-\/render}} extension, place it {\itshape before} the line that calls {\ttfamily auto-\/render.\+js}.
