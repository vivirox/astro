\doxysubsection{Detailed Description}
\href{https://npmjs.com/package/unstorage}{\texttt{ }} \href{https://npmjs.com/package/unstorage}{\texttt{ }} \href{https://codecov.io/gh/unjs/unstorage}{\texttt{ }} \href{https://bundlephobia.com/result?p=unstorage}{\texttt{ }} \href{https://github.com/unjs/unstorage/blob/main/LICENSE}{\texttt{ }}

Unstorage provides an async Key-\/\+Value storage API with conventional features like multi driver mounting, watching and working with metadata, dozens of built-\/in drivers and a \href{https://bundlephobia.com/package/unstorage}{\texttt{ tiny core}}.

\begin{DoxyItemize}
\item Designed for all environments\+: Browser, Node\+JS, and Workers
\item Lots of Built-\/in drivers
\item Asynchronous API
\item Unix-\/style driver mounting to combine storages
\item Default \href{https://unstorage.unjs.io/drivers/memory}{\texttt{ in-\/memory}} storage
\item Tree-\/shakable utils and tiny core
\item Auto JSON value serialization and deserialization
\item Binary and raw value support
\item State \href{https://unstorage.unjs.io/getting-started/utils\#snapshots}{\texttt{ snapshots}} and hydration
\item Storage watcher
\item HTTP Storage with \href{https://unstorage.unjs.io/guide/http-server}{\texttt{ built-\/in server}}
