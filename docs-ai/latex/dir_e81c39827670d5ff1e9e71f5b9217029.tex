\doxysubsection{Detailed Description}
A Node.\+js implementation of the CSS Object Model \href{https://drafts.csswg.org/cssom/\#the-cssstyledeclaration-interface}{\texttt{ {\ttfamily CSSStyle\+Declaration} class}}.\hypertarget{README.md_autotoc_md11192}{}\doxysubsection{\texorpdfstring{Background}{Background}}\label{README.md_autotoc_md11192}
This package is an extension of the {\ttfamily CSSStyle\+Declaration} class in Nikita Vasilyev\textquotesingle{}s \href{https://github.com/NV/CSSOM}{\texttt{ CSSOM}}, with added support for modern specifications. The primary use case is for testing browser code in a Node environment.

It was originally created by Chad Walker, it is now maintained by the jsdom community.

Bug reports and pull requests are welcome.
