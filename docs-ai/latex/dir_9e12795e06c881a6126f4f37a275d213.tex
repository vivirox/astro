This package contains two extensions that add support for math syntax in markdown to \href{https://github.com/syntax-tree/mdast}{\texttt{ mdast}}. These extensions plug into \href{https://github.com/syntax-tree/mdast-util-from-markdown}{\texttt{ {\ttfamily mdast-\/util-\/from-\/markdown}}} (to support parsing math in markdown into a syntax tree) and \href{https://github.com/syntax-tree/mdast-util-to-markdown}{\texttt{ {\ttfamily mdast-\/util-\/to-\/markdown}}} (to support serializing math in syntax trees to markdown).\hypertarget{readme.md_autotoc_md22530}{}\doxysubsection{\texorpdfstring{When to use this}{When to use this}}\label{readme.md_autotoc_md22530}
This project is useful when you want to support math in markdown. Extending markdown with a syntax extension makes the markdown less portable. La\+TeX equations are also quite hard. But this mechanism works well when you want authors, that have some La\+TeX experience, to be able to embed rich diagrams of math in scientific text.

You can use these extensions when you are working with {\ttfamily mdast-\/util-\/from-\/markdown} and {\ttfamily mdast-\/util-\/to-\/markdown} already.

When working with {\ttfamily mdast-\/util-\/from-\/markdown}, you must combine this package with \href{https://github.com/micromark/micromark-extension-math}{\texttt{ {\ttfamily micromark-\/extension-\/math}}}.

When you don’t need a syntax tree, you can use \href{https://github.com/micromark/micromark}{\texttt{ {\ttfamily micromark}}} directly with \href{https://github.com/micromark/micromark-extension-math}{\texttt{ {\ttfamily micromark-\/extension-\/math}}}.

All these packages are used \href{https://github.com/remarkjs/remark-math}{\texttt{ {\ttfamily remark-\/math}}}, which focusses on making it easier to transform content by abstracting these internals away.

This utility adds \href{https://github.com/syntax-tree/mdast-util-to-hast\#fields-on-nodes}{\texttt{ fields on nodes}} so that the utility responsible for turning mdast (markdown) nodes into hast (HTML) nodes, \href{https://github.com/syntax-tree/mdast-util-to-hast}{\texttt{ {\ttfamily mdast-\/util-\/to-\/hast}}}, turns text (inline) math nodes into {\ttfamily \texorpdfstring{$<$}{<}code class="{}language-\/math math-\/inline"{}\texorpdfstring{$>$}{>}…\texorpdfstring{$<$}{<}/code\texorpdfstring{$>$}{>}} and flow (block) math nodes into {\ttfamily \texorpdfstring{$<$}{<}pre\texorpdfstring{$>$}{>}\texorpdfstring{$<$}{<}code class="{}language-\/math math-\/display"{}\texorpdfstring{$>$}{>}…\texorpdfstring{$<$}{<}/code\texorpdfstring{$>$}{>}\texorpdfstring{$<$}{<}/pre\texorpdfstring{$>$}{>}}.\hypertarget{readme.md_autotoc_md22531}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md22531}
