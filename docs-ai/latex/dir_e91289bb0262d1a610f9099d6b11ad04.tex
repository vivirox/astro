\doxysubsection{Detailed Description}
JSON language service extracted from VSCode to be reused, e.\+g in the Monaco editor.

\href{https://www.npmjs.org/package/vscode-json-languageservice}{\texttt{ }} \href{https://npmjs.org/package/vscode-json-languageservice}{\texttt{ }} \href{https://travis-ci.org/Microsoft/vscode-json-languageservice}{\texttt{ }} \href{https://opensource.org/licenses/MIT}{\texttt{ }}\hypertarget{README.md_autotoc_md35371}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{README.md_autotoc_md35371}
The {\itshape vscode-\/json-\/languageservice} contains the language smarts behind the JSON editing experience of Visual Studio Code and the Monaco editor.


\begin{DoxyItemize}
\item {\itshape do\+Validation} analyses an input string and returns syntax and lint errors.
\item {\itshape do\+Complete} provides completion proposals for a given location.
\item {\itshape do\+Resolve} resolves a completion proposals.
\item {\itshape do\+Hover} provides a hover text for a given location.
\item {\itshape find\+Document\+Symbols} provides all symbols in the given document.
\item {\itshape find\+Document\+Colors} provides all color symbols in the given document.
\item {\itshape get\+Color\+Presentations} returns available color formats for a color symbol.
\item {\itshape format} formats the code at the given range.
\item {\itshape get\+Folding\+Ranges} gets folding ranges for the given document.
\item {\itshape get\+Selection\+Ranges} gets selection ranges for a given location.
\item {\itshape get\+Matching\+Schemas} matches a document against its schema and returns all AST nodes along with the matching sub schemas.
\item {\itshape parse\+JSONDocument} creates a JSON document from source code.
\item {\itshape new\+JSONDocument} creates a JSON document from an AST.
\end{DoxyItemize}
