\doxysubsection{Detailed Description}
A library to create readable {\ttfamily "{}multipart/form-\/data"{}} streams. Can be used to submit forms and file uploads to other web applications.

The API of this library is inspired by the \href{http://dev.w3.org/2006/webapi/XMLHttpRequest-2/Overview.html\#the-formdata-interface}{\texttt{ XMLHttp\+Request-\/2 Form\+Data Interface}}.

\hypertarget{Readme.md_autotoc_md15789}{}\doxysubsection{\texorpdfstring{Notes}{Notes}}\label{Readme.md_autotoc_md15789}

\begin{DoxyItemize}
\item {\ttfamily get\+Length\+Sync()} method DOESN\textquotesingle{}T calculate length for streams, use {\ttfamily known\+Length} options as workaround.
\item {\ttfamily get\+Length(cb)} will send an error as first parameter of callback if stream length cannot be calculated (e.\+g. send in custom streams w/o using {\ttfamily known\+Length}).
\item {\ttfamily submit} will not add {\ttfamily content-\/length} if form length is unknown or not calculable.
\item Starting version {\ttfamily 2.\+x} Form\+Data has dropped support for {\ttfamily node@0.\+10.\+x}.
\item Starting version {\ttfamily 3.\+x} Form\+Data has dropped support for {\ttfamily node@4.\+x}.
\end{DoxyItemize}\hypertarget{Readme.md_autotoc_md15790}{}\doxysubsection{\texorpdfstring{License}{License}}\label{Readme.md_autotoc_md15790}
