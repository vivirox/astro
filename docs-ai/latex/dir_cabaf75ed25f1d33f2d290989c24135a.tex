\doxysubsection{Detailed Description}
Library used for loading the translations into subprocesses of your extension. These usages also get picked up by \href{https://github.com/microsoft/vscode-l10n/tree/main/l10n-dev}{\texttt{ l10n-\/dev}} string extraction tooling.

\begin{quote}
{\bfseries{Note}}

You should {\itshape NOT} use this library in your extension\textquotesingle{}s main process. The translations are loaded into the main process by VS Code itself. \end{quote}

The input for {\ttfamily l10n.\+config} pairs nicely with the {\ttfamily bundle} and {\ttfamily uri} properties on the {\ttfamily l10n} namespace that are provided by the \href{https://code.visualstudio.com/api/references/vscode-api\#l10n}{\texttt{ VS Code API}}. You should send the value of one of these properties from your extension to your subprocess that is consuming {\ttfamily @vscode/l10n}.
