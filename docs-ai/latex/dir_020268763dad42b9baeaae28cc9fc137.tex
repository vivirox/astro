\doxysubsection{Detailed Description}
\begin{quote}
Convert Windows backslash paths to slash paths\+: {\ttfamily foo\textbackslash{}\textbackslash{}bar} ➔ {\ttfamily foo/bar} \end{quote}


\href{http://superuser.com/a/176395/6877}{\texttt{ Forward-\/slash paths can be used in Windows}} as long as they\textquotesingle{}re not extended-\/length paths and don\textquotesingle{}t contain any non-\/ascii characters.

This was created since the {\ttfamily path} methods in Node.\+js outputs {\ttfamily \textbackslash{}\textbackslash{}} paths on Windows.\hypertarget{readme.md_autotoc_md30864}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md30864}
