For every {\ttfamily IsT} type (e.\+g. {\ttfamily Is\+Any}), there is an associated {\ttfamily IfT} type that can help simplify conditional types. While the {\ttfamily IsT} types return a {\ttfamily boolean}, the {\ttfamily IfT} types act like an {\ttfamily If}/{\ttfamily Else} -\/ they resolve to the given {\ttfamily Type\+IfT} or {\ttfamily Type\+If\+NotT} depending on whether {\ttfamily IsX} is {\ttfamily true} or not. By default, {\ttfamily IfT} returns a {\ttfamily boolean}\+:
