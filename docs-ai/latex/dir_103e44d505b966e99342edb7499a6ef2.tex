When calling {\ttfamily debounced}, it will wait at least for {\ttfamily 25ms} as configured before actually calling our function. This helps to avoid multiple calls.

To avoid initial wait, we can set {\ttfamily leading\+: true} option. It will cause function to be immediately called if there is no other call\+:
