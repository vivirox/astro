This package contains lots of info on all the properties and attributes found on the web platform. It includes data on HTML, SVG, ARIA, XML, XMLNS, and XLink. The names of the properties follow \href{https://github.com/syntax-tree/hast\#propertyname}{\texttt{ hast}}’s sensible naming scheme. It includes info on what data types attributes hold, such as whether they’re booleans or contain lists of space separated numbers.\hypertarget{readme.md_autotoc_md17512}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md17512}
You can use this package if you’re working with hast, which is an AST for HTML, or have goals related to ASTs, such as figuring out which properties or attributes are valid, or what data types they hold.\hypertarget{readme.md_autotoc_md17513}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md17513}

There is no default export.\hypertarget{readme.md_autotoc_md17516}{}\doxysubsubsection{\texorpdfstring{{\ttfamily find(schema, name)}}{{\ttfamily find(schema, name)}}}\label{readme.md_autotoc_md17516}
Look up info on a property.

In most cases, the given {\ttfamily schema} contains info on the property. All standard, most legacy, and some non-\/standard properties are supported. For these cases, the returned \`{}\+Info\`{} has hints about the value of the property.

{\ttfamily name} can also be a \href{https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement/dataset}{\texttt{ valid data attribute or property}}, in which case an \`{}\+Info\`{} object with the correctly cased {\ttfamily attribute} and {\ttfamily property} is returned.

{\ttfamily name} can be an unknown attribute, in which case an \`{}\+Info\`{} object with {\ttfamily attribute} and {\ttfamily property} set to the given name is returned. It is not recommended to provide unsupported legacy or recently specced properties.\hypertarget{readme.md_autotoc_md17517}{}\doxysubsubsubsection{\texorpdfstring{Parameters}{Parameters}}\label{readme.md_autotoc_md17517}
