\doxysubsection{Detailed Description}
\href{http://badge.fury.io/js/escodegen}{\texttt{ }} \href{http://travis-ci.org/estools/escodegen}{\texttt{ }} \href{https://david-dm.org/estools/escodegen}{\texttt{ }} \href{https://david-dm.org/estools/escodegen\#info=devDependencies}{\texttt{ }}

Escodegen (\href{http://github.com/estools/escodegen}{\texttt{ escodegen}}) is an \href{http://www.ecma-international.org/publications/standards/Ecma-262.htm}{\texttt{ ECMAScript}} (also popularly known as \href{http://en.wikipedia.org/wiki/JavaScript}{\texttt{ Java\+Script}}) code generator from \href{https://developer.mozilla.org/en/SpiderMonkey/Parser_API}{\texttt{ Mozilla\textquotesingle{}s Parser API}} AST. See the \href{https://estools.github.io/escodegen/demo/index.html}{\texttt{ online generator}} for a demo.\hypertarget{README.md_autotoc_md12903}{}\doxysubsubsection{\texorpdfstring{Install}{Install}}\label{README.md_autotoc_md12903}
Escodegen can be used in a web browser\+: \begin{DoxyVerb}<script src="escodegen.browser.js"></script>
\end{DoxyVerb}


escodegen.\+browser.\+js can be found in tagged revisions on Git\+Hub.

Or in a Node.\+js application via npm\+: \begin{DoxyVerb}npm install escodegen
\end{DoxyVerb}
\hypertarget{README.md_autotoc_md12904}{}\doxysubsubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md12904}
A simple example\+: the program \begin{DoxyVerb}escodegen.generate({
    type: 'BinaryExpression',
    operator: '+',
    left: { type: 'Literal', value: 40 },
    right: { type: 'Literal', value: 2 }
});
\end{DoxyVerb}


produces the string `\textquotesingle{}40 + 2'\`{}.

See the \href{https://github.com/estools/escodegen/wiki/API}{\texttt{ API page}} for options. To run the tests, execute {\ttfamily npm test} in the root directory.\hypertarget{README.md_autotoc_md12905}{}\doxysubsubsection{\texorpdfstring{Building browser bundle / minified browser bundle}{Building browser bundle / minified browser bundle}}\label{README.md_autotoc_md12905}
At first, execute {\ttfamily npm install} to install the all dev dependencies. After that, \begin{DoxyVerb}npm run-script build
\end{DoxyVerb}


will generate {\ttfamily escodegen.\+browser.\+js}, which can be used in browser environments.

And, \begin{DoxyVerb}npm run-script build-min
\end{DoxyVerb}


will generate the minified file {\ttfamily escodegen.\+browser.\+min.\+js}.\hypertarget{README.md_autotoc_md12906}{}\doxysubsubsection{\texorpdfstring{License}{License}}\label{README.md_autotoc_md12906}
