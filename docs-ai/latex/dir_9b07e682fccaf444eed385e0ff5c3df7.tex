\doxysubsection{Detailed Description}
We welcome any localization for axe-\/core. For details on how to contribute, see the \href{../README.md\#contributing}{\texttt{ Contributing section}} of the main README. For details on the message syntax, see Check Message Template.

To create a new translation for axe, start by running {\ttfamily grunt translate -\/-\/lang=\texorpdfstring{$<$}{<}langcode\texorpdfstring{$>$}{>}}. This will create a JSON file with the default English text in it for you to translate. Alternatively, you could copy {\ttfamily \+\_\+template.\+json}.

To update an existing translation file, re-\/run {\ttfamily grunt translate -\/-\/lang=\texorpdfstring{$<$}{<}langcode\texorpdfstring{$>$}{>}}. This will add new messages used in English and remove messages that are no longer used in English.

{\ttfamily \+\_\+template.\+json} is a generated file which is created every time axe is built. It\textquotesingle{}s compiled using each rule\textquotesingle{}s {\ttfamily description} and {\ttfamily help} properties, as well as each check\textquotesingle{}s {\ttfamily metadata.\+messages} property. To update the {\ttfamily \+\_\+template.\+json} file you\textquotesingle{}ll need to update the corresponding \href{../lib/rules}{\texttt{ rule}} or \href{../lib/checks}{\texttt{ check}} metadata file and rebuild.
