Lint {\ttfamily .md} files with {\ttfamily eslint-\/plugin-\/markdown}.

By default, {\ttfamily eslint-\/plugin-\/markdown}\textquotesingle{}s processor will create a virtual file for each code snippet in the markdown file, {\bfseries{but not the original {\ttfamily .md} file itself}}. This means the original {\ttfamily .md} file will not be linted.

With this package, along with the {\ttfamily processor\+Pass\+Through} processor, you can now fix that\+:
