This package applies some transforms to a cloned, given estree to make it compatible with unist. It\+:


\begin{DoxyItemize}
\item makes sure nodes are plain JSON
\item adds unist positions
\item normalizes {\ttfamily .bigint}
\item remove {\ttfamily attributes}, {\ttfamily self\+Closing} from {\ttfamily JSXOpening\+Fragment}
\item removes certain discouraged fields
\end{DoxyItemize}\hypertarget{readme.md_autotoc_md12735}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md12735}
The transform applied by this utility is often optional\+: estrees can be used in most places where esast can be used, and vice versa.
