\hypertarget{README.md_autotoc_md3744}{}\doxysubsubsection{\texorpdfstring{Theming}{Theming}}\label{README.md_autotoc_md3744}
You can theme a prompt by passing a {\ttfamily theme} object option. The theme object only need to includes the keys you wish to modify, we\textquotesingle{}ll fallback on the defaults for the rest.


\begin{DoxyCode}{0}
\DoxyCodeLine{type\ Theme\ =\ \{}
\DoxyCodeLine{\ \ prefix:\ string\ |\ \{\ idle:\ string;\ done:\ string\ \};}
\DoxyCodeLine{\ \ spinner:\ \{}
\DoxyCodeLine{\ \ \ \ interval:\ number;}
\DoxyCodeLine{\ \ \ \ frames:\ string[];}
\DoxyCodeLine{\ \ \};}
\DoxyCodeLine{\ \ style:\ \{}
\DoxyCodeLine{\ \ \ \ message:\ (text:\ string,\ status:\ 'idle'\ |\ 'done'\ |\ 'loading')\ =>\ string;}
\DoxyCodeLine{\ \ \ \ error:\ (text:\ string)\ =>\ string;}
\DoxyCodeLine{\ \ \ \ help:\ (text:\ string)\ =>\ string;}
\DoxyCodeLine{\ \ \ \ key:\ (text:\ string)\ =>\ string;}
\DoxyCodeLine{\ \ \};}
\DoxyCodeLine{\ \ validationFailureMode:\ 'keep'\ |\ 'clear';}
\DoxyCodeLine{\};}

\end{DoxyCode}


{\ttfamily validation\+Failure\+Mode} defines the behavior of the prompt when the value submitted is invalid. By default, we\textquotesingle{}ll keep the value allowing the user to edit it.
