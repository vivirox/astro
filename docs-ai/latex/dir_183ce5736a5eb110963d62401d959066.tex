This repository provides a mapping of Electron versions to the Chromium version that it uses.

This package is used in \href{https://github.com/ai/browserslist}{\texttt{ Browserslist}}, so you can use e.\+g. {\ttfamily electron \texorpdfstring{$>$}{>}= 1.\+4} in \href{https://github.com/postcss/autoprefixer}{\texttt{ Autoprefixer}}, \href{https://github.com/stylelint/stylelint}{\texttt{ Stylelint}}, \href{https://github.com/babel/babel-preset-env}{\texttt{ babel-\/preset-\/env}} and \href{https://github.com/amilajack/eslint-plugin-compat}{\texttt{ eslint-\/plugin-\/compat}}.
