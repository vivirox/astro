\doxysubsection{Detailed Description}
\begin{quote}
Safely get a dot-\/notated path within a nested object, with ability to return a default if the full key path does not exist or the value is undefined \end{quote}
\hypertarget{README.md_autotoc_md12005}{}\doxysubsubsection{\texorpdfstring{Why?}{Why?}}\label{README.md_autotoc_md12005}
Smallest possible implementation\+: only {\bfseries{130 bytes.}}

You could write this yourself, but then you\textquotesingle{}d have to write \href{https://github.com/developit/dlv/blob/master/test.js}{\texttt{ tests}}.

Supports ES Modules, Common\+JS and globals.\hypertarget{README.md_autotoc_md12006}{}\doxysubsubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md12006}
