\doxysubsection{Detailed Description}
Parses given Emmet {\itshape markup} abbreviation into AST. Parsing is performed in two steps\+: first it tokenizes given abbreviation (useful for syntax highlighting in editors) and then tokens are analyzed and converted into AST nodes as plain, JSON-\/serializable objects.

Note that AST tree in most cases cannot be used directly for output\+: for example, AST node produced from {\ttfamily .foo.\+bar} element misses element name and contains two {\ttfamily class} attributes with {\ttfamily foo} and {\ttfamily bar} values (not a single {\ttfamily class} with {\ttfamily foo bar} value).\hypertarget{README.md_autotoc_md2996}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md2996}
You can install it via npm\+:


\begin{DoxyCode}{0}

 The returned tree contains {\ttfamily Abbreviation\+Node} items\+: a node with name, attributes and/or text content. E.\+g. an element that can be represented somehow. Repeated and grouped nodes like {\ttfamily a\texorpdfstring{$>$}{>}(b+c)\texorpdfstring{$\ast$}{*}3} are automatically converted and duplicated as distinct {\ttfamily Abbreviation\+Node} with distinct {\ttfamily .repeat} property which identifies node in repeating sequence.\hypertarget{README.md_autotoc_md2997}{}\doxysubsection{\texorpdfstring{Abbreviation syntax}{Abbreviation syntax}}\label{README.md_autotoc_md2997}

Each element of abbreviation must be separated with any of these operators\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{elem1+elem2>elem3}

\end{DoxyCode}



\begin{DoxyItemize}
\item {\ttfamily +} — sibling operator, adds next element as a next sibling of current element in tree.
\item {\ttfamily \texorpdfstring{$>$}{>}} — child operator, adds next element as a child of current element.
\item {\ttfamily \texorpdfstring{$^\wedge$}{\string^}} — climb-\/up operator, adds next element as a child of current element’s parent node. Multiple climb-\/up operators are allowed, each operator moves one level up by tree.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md2999}{}\doxysubsubsection{\texorpdfstring{Groups}{Groups}}\label{README.md_autotoc_md2999}
