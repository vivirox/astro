\href{https://npmjs.com/package/eslint-plugin-pnpm}{\texttt{ }} \href{https://npmjs.com/package/eslint-plugin-pnpm}{\texttt{ }} \href{https://bundlephobia.com/result?p=eslint-plugin-pnpm}{\texttt{ }} \href{https://www.jsdocs.io/package/eslint-plugin-pnpm}{\texttt{ }} \href{https://github.com/antfu/eslint-plugin-pnpm/blob/main/LICENSE}{\texttt{ }}

ESLint plugin to enforce and auto-\/fix pnpm catalogs.

This plugin consist with two set of rules that applies to {\ttfamily package.\+json} and {\ttfamily pnpm-\/workspace.\+yaml} respectively.


\begin{DoxyItemize}
\item \href{./src/rules/json}{\texttt{ {\ttfamily json-\/} rules}} applies to {\ttfamily package.\+json} and requires \href{https://github.com/ota-meshi/jsonc-eslint-parser}{\texttt{ {\ttfamily jsonc-\/eslint-\/parser}}} to be used as parser.
\item \href{./src/rules/yaml}{\texttt{ {\ttfamily yaml-\/} rules}} applies to {\ttfamily pnpm-\/workspace.\+yaml} and requires \href{https://github.com/ota-meshi/yaml-eslint-parser}{\texttt{ {\ttfamily yaml-\/eslint-\/parser}}} to be used as parser.
\begin{DoxyItemize}
\item YAML support is still experimental as it might have race conditions with other plugins.
\end{DoxyItemize}
