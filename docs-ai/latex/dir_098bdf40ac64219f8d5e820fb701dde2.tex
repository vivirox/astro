\doxysubsection{Detailed Description}
This directory contains translation data for Starlight’s UI. Each language has its own JSON file and follows the \href{https://starlight.astro.build/guides/i18n/\#translate-starlights-ui}{\texttt{ translation structure described in Starlight’s docs}}.\hypertarget{README.md_autotoc_md1579}{}\doxysubsection{\texorpdfstring{Add a new language}{Add a new language}}\label{README.md_autotoc_md1579}

\begin{DoxyEnumerate}
\item Create a JSON file named using the BCP-\/47 tag for the language, e.\+g. {\ttfamily en.\+json} or {\ttfamily ja.\+json}.
\item Fill the file with translations for each UI string. You can base your translations on \href{./en.json}{\texttt{ {\ttfamily en.\+json}}}. Translate only the values, leaving the keys in English (e.\+g. {\ttfamily "{}search.\+label"{}\+: "{}\+Buscar"{}}).
\item Import your file in \href{./index.ts}{\texttt{ {\ttfamily index.\+ts}}} and add your language to the {\ttfamily Object.\+entries}.
\item Open a pull request on Git\+Hub to add your file to Starlight! 
\end{DoxyEnumerate}
