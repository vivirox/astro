 A tiny (2.\+8kB) and fast utility for getting a MIME type from an extension or filename \hypertarget{readme.md_autotoc_md24383}{}\doxysubsubsection{\texorpdfstring{Features}{Features}}\label{readme.md_autotoc_md24383}

\begin{DoxyItemize}
\item Lightweight – 2.\+8kB gzip~\newline
 {\itshape Only includes standard mime types; all experimental and vendor-\/specific mimetypes removed.}
\item Performant~\newline
 {\itshape All lookups are O(1) with minimal processing.}
\item Comprehensive Dictionary~\newline
 {\itshape Generated from \href{https://github.com/jshttp/mime-db}{\texttt{ {\ttfamily mime-\/db}}}, which aggregates the IANA, NGINX, and Apache datasets.}
\item Customizable~\newline
 {\itshape Exposes the {\ttfamily mimes} dictionary for easy additions or overrides.}
\item Supports Native ESM and \href{https://deno.land/x/mrmime}{\texttt{ Deno}}~\newline
 {\itshape Ships with Common\+JS and ESM support!}
