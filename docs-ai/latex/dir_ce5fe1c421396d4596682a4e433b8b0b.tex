\doxysubsection{Detailed Description}
\begin{quote}
Convert a dash/dot/underscore/space separated string to camel\+Case or Pascal\+Case\+: {\ttfamily foo-\/bar} → {\ttfamily foo\+Bar} \end{quote}


Correctly handles Unicode strings.

If you use this on untrusted user input, don\textquotesingle{}t forget to limit the length to something reasonable.\hypertarget{readme.md_autotoc_md9798}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md9798}

Preserve consecutive uppercase characters\+: {\ttfamily foo-\/\+BAR} → {\ttfamily Foo\+BAR}.\hypertarget{readme.md_autotoc_md9806}{}\doxysubsubsubsubsection{\texorpdfstring{locale}{locale}}\label{readme.md_autotoc_md9806}

The locale parameter indicates the locale to be used to convert to upper/lower case according to any locale-\/specific case mappings. If multiple locales are given in an array, the best available locale is used.
