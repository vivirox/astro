An ESnext spec-\/compliant {\ttfamily Array.\+prototype.\+to\+Sorted} shim/polyfill/replacement that works as far down as ES3.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the proposed \href{https://tc39.es/proposal-change-array-by-copy/\#sec-array.prototype.toSorted}{\texttt{ spec}}.
