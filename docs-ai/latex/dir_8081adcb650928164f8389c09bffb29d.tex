\doxysubsection{Detailed Description}
\begin{quote}
Generate source maps \end{quote}


{\ttfamily gen-\/mapping} allows you to generate a source map during transpilation or minification. With a source map, you\textquotesingle{}re able to trace the original location in the source file, either in Chrome\textquotesingle{}s Dev\+Tools or using a library like \href{https://github.com/jridgewell/trace-mapping}{\texttt{ {\ttfamily @jridgewell/trace-\/mapping}}}.

You may already be familiar with the \href{https://www.npmjs.com/package/source-map}{\texttt{ {\ttfamily source-\/map}}} package\textquotesingle{}s {\ttfamily Source\+Map\+Generator}. This provides the same {\ttfamily add\+Mapping} and {\ttfamily set\+Source\+Content} API.\hypertarget{README.md_autotoc_md4659}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md4659}
