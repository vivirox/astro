An ECMAScript spec-\/compliant polyfill/shim for {\ttfamily global\+This}. Invoke its "{}shim"{} method to shim {\ttfamily global\+This} if it is unavailable.

This package implements the \href{https://github.com/es-shims/api}{\texttt{ es-\/shim API}} interface. It works in an ES3-\/supported environment and complies with the \href{https://github.com/tc39/proposal-global}{\texttt{ spec proposal}}.
