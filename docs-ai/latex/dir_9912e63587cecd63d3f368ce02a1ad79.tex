This package exposes constants used throughout the micromark ecosystem.\hypertarget{readme.md_autotoc_md23806}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md23806}
This package is useful when you are making your own micromark extensions. It’s useful to reference these constants by name instead of value while developing. \href{https://github.com/micromark/micromark/tree/main/packages/micromark-build}{\texttt{ {\ttfamily micromark-\/build}}} compiles them away for production code.\hypertarget{readme.md_autotoc_md23807}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md23807}
