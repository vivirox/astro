\doxysubsection{Detailed Description}
Display differences clearly so people can review changes confidently.

The {\ttfamily diff} named export serializes Java\+Script {\bfseries{values}}, compares them line-\/by-\/line, and returns a string which includes comparison lines.

Two named exports compare {\bfseries{strings}} character-\/by-\/character\+:


\begin{DoxyItemize}
\item {\ttfamily diff\+Strings\+Unified} returns a string.
\item {\ttfamily diff\+Strings\+Raw} returns an array of {\ttfamily Diff} objects.
\end{DoxyItemize}

Three named exports compare {\bfseries{arrays of strings}} line-\/by-\/line\+:


\begin{DoxyItemize}
\item {\ttfamily diff\+Lines\+Unified} and {\ttfamily diff\+Lines\+Unified2} return a string.
\item {\ttfamily diff\+Lines\+Raw} returns an array of {\ttfamily Diff} objects.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md19816}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md19816}
