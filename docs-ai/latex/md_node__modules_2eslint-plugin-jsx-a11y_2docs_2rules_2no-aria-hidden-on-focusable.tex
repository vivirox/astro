Enforce that {\ttfamily aria-\/hidden="{}true"{}} is not set on focusable elements.

{\ttfamily aria-\/hidden="{}true"{}} can be used to hide purely decorative content from screen reader users. An element with {\ttfamily aria-\/hidden="{}true"{}} that can also be reached by keyboard can lead to confusion or unexpected behavior for screen reader users. Avoid using {\ttfamily aria-\/hidden="{}true"{}} on focusable elements.\hypertarget{md_node__modules_2eslint-plugin-jsx-a11y_2docs_2rules_2no-aria-hidden-on-focusable_autotoc_md13880}{}\doxysection{\texorpdfstring{Rule details}{Rule details}}\label{md_node__modules_2eslint-plugin-jsx-a11y_2docs_2rules_2no-aria-hidden-on-focusable_autotoc_md13880}
