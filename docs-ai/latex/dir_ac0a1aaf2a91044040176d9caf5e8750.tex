\doxysubsection{Detailed Description}
HTML language service extracted from VSCode to be reused, e.\+g in the Monaco editor.

\href{https://www.npmjs.org/package/vscode-html-languageservice}{\texttt{ }} \href{https://npmjs.org/package/vscode-html-languageservice}{\texttt{ }} \href{https://github.com/microsoft/vscode-html-languageservice/actions}{\texttt{ }} \href{https://opensource.org/licenses/MIT}{\texttt{ }}\hypertarget{README.md_autotoc_md35307}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{README.md_autotoc_md35307}
The {\itshape vscode-\/html-\/languageservice} contains the language smarts behind the HTML editing experience of Visual Studio Code and the Monaco editor.


\begin{DoxyItemize}
\item {\itshape do\+Complete} / {\itshape do\+Complete2} (async) provide completion proposals for a given location.
\item {\itshape set\+Completion\+Participants} allows participant to provide suggestions for specific tokens.
\item {\itshape do\+Hover} provides hover information at a given location.
\item {\itshape format} formats the code at the given range.
\item {\itshape find\+Document\+Links} finds all links in the document.
\item {\itshape find\+Document\+Symbols} finds all the symbols in the document.
\item {\itshape get\+Folding\+Ranges} return folding ranges for the given document.
\item {\itshape get\+Selection\+Ranges} return the selection ranges for the given document. ...
\end{DoxyItemize}
