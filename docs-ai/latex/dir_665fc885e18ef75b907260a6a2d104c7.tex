\doxysubsection{Detailed Description}
\doxysubsection*{\href{https://katex.org/}{\texttt{   }}}

\href{https://www.npmjs.com/package/katex}{\texttt{ }} \href{https://github.com/semantic-release/semantic-release}{\texttt{ }} \href{https://github.com/KaTeX/KaTeX/actions?query=workflow\%3ACI}{\texttt{ }} \href{https://codecov.io/gh/KaTeX/KaTeX}{\texttt{ }} \href{https://github.com/KaTeX/KaTeX/discussions}{\texttt{ }} \href{https://www.jsdelivr.com/package/npm/katex}{\texttt{ }}  \href{https://gitpod.io/\#https://github.com/KaTeX/KaTeX}{\texttt{ }} \href{https://opencollective.com/katex}{\texttt{ }}

Ka\+TeX is a fast, easy-\/to-\/use Java\+Script library for TeX math rendering on the web.


\begin{DoxyItemize}
\item {\bfseries{Fast\+:}} Ka\+TeX renders its math synchronously and doesn\textquotesingle{}t need to reflow the page. See how it compares to a competitor in \href{https://www.intmath.com/cg5/katex-mathjax-comparison.php}{\texttt{ this speed test}}.
\item {\bfseries{Print quality\+:}} Ka\+TeX\textquotesingle{}s layout is based on Donald Knuth\textquotesingle{}s TeX, the gold standard for math typesetting.
\item {\bfseries{Self contained\+:}} Ka\+TeX has no dependencies and can easily be bundled with your website resources.
\item {\bfseries{Server side rendering\+:}} Ka\+TeX produces the same output regardless of browser or environment, so you can pre-\/render expressions using Node.\+js and send them as plain HTML.
\end{DoxyItemize}

Ka\+TeX is compatible with all major browsers, including Chrome, Safari, Firefox, Opera, Edge, and IE 11.

Ka\+TeX supports much (but not all) of La\+TeX and many La\+TeX packages. See the \href{https://katex.org/docs/supported.html}{\texttt{ list of supported functions}}.
