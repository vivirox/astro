Modifies the {\ttfamily to} function to mimic the {\ttfamily from} function. Returns the {\ttfamily to} function.

{\ttfamily name}, {\ttfamily display\+Name}, and any other properties of {\ttfamily from} are copied. The {\ttfamily length} property is not copied. Prototype, class, and inherited properties are copied.
