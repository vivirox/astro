\doxysubsection{Detailed Description}
Utility for determining whether an AST node is a reference.

{\ttfamily foo} is a reference in these cases\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(foo);}
\DoxyCodeLine{var\ foo;}
\DoxyCodeLine{function\ foo()\ \{\}}
\DoxyCodeLine{function\ bar(foo)\ \{\}}
\DoxyCodeLine{export\ \{\ foo\ as\ x\ \};}

\end{DoxyCode}


{\ttfamily foo} is {\itshape not} a reference in these cases\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ obj\ =\ \{\ foo:\ 1\ \};}
\DoxyCodeLine{console.log(obj.foo);}
\DoxyCodeLine{export\ \{\ x\ as\ foo\ \};}

\end{DoxyCode}


In all cases, {\ttfamily foo} is an {\ttfamily Identifier} node, but the two kinds must be treated differently for the purposes of scope analysis etc. (The examples are non-\/exhaustive.)\hypertarget{README.md_autotoc_md18727}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md18727}

\end{DoxyCode}
