{\ttfamily }\hypertarget{README.md_autotoc_md10951}{}\doxysubsection{\texorpdfstring{{\ttfamily Log Level}}{{\ttfamily Log Level}}}\label{README.md_autotoc_md10951}
{\ttfamily }

{\ttfamily Consola only shows logs with configured log level or below. (Default is {\ttfamily 3})}

{\ttfamily Available log levels\+:}

{\ttfamily 
\begin{DoxyItemize}
\item {\ttfamily 0}\+: Fatal and Error
\item {\ttfamily 1}\+: Warnings
\item {\ttfamily 2}\+: Normal logs
\item {\ttfamily 3}\+: Informational logs, success, fail, ready, start, ...
\item {\ttfamily 4}\+: Debug logs
\item {\ttfamily 5}\+: Trace logs
\item {\ttfamily -\/999}\+: Silent
\item {\ttfamily +999}\+: Verbose logs
\end{DoxyItemize}}

{\ttfamily You can set the log level by either\+:}

{\ttfamily 
\begin{DoxyItemize}
\item Passing {\ttfamily level} option to {\ttfamily create\+Consola}
\item Setting {\ttfamily consola.\+level} on instance
\item Using the {\ttfamily CONSOLA\+\_\+\+LEVEL} environment variable (not supported for browser and core builds).
\end{DoxyItemize}}
