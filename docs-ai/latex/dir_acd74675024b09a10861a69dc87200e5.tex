\hypertarget{README.md_autotoc_md10310}{}\doxysubsubsection{\texorpdfstring{{\ttfamily ci.\+name}}{{\ttfamily ci.\+name}}}\label{README.md_autotoc_md10310}
Returns a string containing name of the CI server the code is running on. If CI server is not detected, it returns {\ttfamily null}.

Don\textquotesingle{}t depend on the value of this string not to change for a specific vendor. If you find your self writing `ci.\+name === \textquotesingle{}Travis CI'{\ttfamily , you most likely want to use}ci.\+TRAVIS\`{} instead.\hypertarget{README.md_autotoc_md10311}{}\doxysubsubsection{\texorpdfstring{{\ttfamily ci.\+is\+CI}}{{\ttfamily ci.\+is\+CI}}}\label{README.md_autotoc_md10311}
Returns a boolean. Will be {\ttfamily true} if the code is running on a CI server, otherwise {\ttfamily false}.

Some CI servers not listed here might still trigger the {\ttfamily ci.\+is\+CI} boolean to be set to {\ttfamily true} if they use certain vendor neutral environment variables. In those cases {\ttfamily ci.\+name} will be {\ttfamily null} and no vendor specific boolean will be set to {\ttfamily true}.\hypertarget{README.md_autotoc_md10312}{}\doxysubsubsection{\texorpdfstring{{\ttfamily ci.\+is\+PR}}{{\ttfamily ci.\+is\+PR}}}\label{README.md_autotoc_md10312}
Returns a boolean if PR detection is supported for the current CI server. Will be {\ttfamily true} if a PR is being tested, otherwise {\ttfamily false}. If PR detection is not supported for the current CI server, the value will be {\ttfamily null}.\hypertarget{README.md_autotoc_md10313}{}\doxysubsubsection{\texorpdfstring{{\ttfamily ci.\texorpdfstring{$<$}{<}VENDOR-\/\+CONSTANT\texorpdfstring{$>$}{>}}}{{\ttfamily ci.\texorpdfstring{$<$}{<}VENDOR-\/\+CONSTANT\texorpdfstring{$>$}{>}}}}\label{README.md_autotoc_md10313}
A vendor specific boolean constant is exposed for each support CI vendor. A constant will be {\ttfamily true} if the code is determined to run on the given CI server, otherwise {\ttfamily false}.

Examples of vendor constants are {\ttfamily ci.\+TRAVIS} or {\ttfamily ci.\+APPVEYOR}. For a complete list, see the support table above.\hypertarget{README.md_autotoc_md10314}{}\doxysubsection{\texorpdfstring{Ports}{Ports}}\label{README.md_autotoc_md10314}
