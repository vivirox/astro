\doxysubsection{Detailed Description}
Simple Javascript logger for Node.\+js and Browsers

{\bfseries{\href{https://ipfs.io/ipfs/QmRrBe2sp9ha2xypRoz5UDXqBJUB83NcecQU3QpqBJ5hkq}{\texttt{ DEMO}}}} {\itshape Open the dev tools to see the log output}



{\itshape logplease} does two simple things\+: output log messages to the console and/or to a file (Node.\+js only) and display the log messages with nice colors. Inspired by \href{https://github.com/stritti/log4js}{\texttt{ log4js}} and \href{https://github.com/visionmedia/debug}{\texttt{ debug}}.\hypertarget{README.md_autotoc_md22041}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md22041}

\begin{DoxyItemize}

\item Log levels
\item Colors!
\item Work in Node.\+js and Browsers
