Create a virtual file. Works like the \href{https://github.com/vfile/vfile}{\texttt{ vfile}} constructor, except when {\ttfamily options} is {\ttfamily string} or {\ttfamily Buffer}, in which case it’s treated as {\ttfamily \{path\+: options\}} instead of {\ttfamily \{contents\+: options\}}.\hypertarget{readme.md_autotoc_md33017}{}\doxysubsubsection{\texorpdfstring{{\ttfamily to\+VFile.\+read(options\mbox{[}, encoding\mbox{]}\mbox{[}, callback\mbox{]})}}{{\ttfamily to\+VFile.\+read(options\mbox{[}, encoding\mbox{]}\mbox{[}, callback\mbox{]})}}}\label{readme.md_autotoc_md33017}
Creates a virtual file from options ({\ttfamily to\+VFile(options)}), reads the file from the file-\/system and populates {\ttfamily file.\+contents} with the result. If {\ttfamily encoding} is specified, it’s passed to {\ttfamily fs.\+read\+File}. If {\ttfamily callback} is given, invokes it with either an error or the populated virtual file. If {\ttfamily callback} is not given, returns a \href{https://developer.mozilla.org/Web/JavaScript/Reference/Global_Objects/Promise}{\texttt{ {\ttfamily Promise}}} that is rejected with an error or resolved with the populated virtual file.\hypertarget{readme.md_autotoc_md33018}{}\doxysubsubsection{\texorpdfstring{{\ttfamily to\+VFile.\+read\+Sync(options\mbox{[}, encoding\mbox{]})}}{{\ttfamily to\+VFile.\+read\+Sync(options\mbox{[}, encoding\mbox{]})}}}\label{readme.md_autotoc_md33018}
Like {\ttfamily to\+VFile.\+read} but synchronous. Either throws an error or returns a populated virtual file.\hypertarget{readme.md_autotoc_md33019}{}\doxysubsubsection{\texorpdfstring{{\ttfamily to\+VFile.\+write(options\mbox{[}, fs\+Options\mbox{]}\mbox{[}, callback\mbox{]})}}{{\ttfamily to\+VFile.\+write(options\mbox{[}, fs\+Options\mbox{]}\mbox{[}, callback\mbox{]})}}}\label{readme.md_autotoc_md33019}
Creates a virtual file from {\ttfamily options} ({\ttfamily to\+VFile(options)}), writes the file to the file-\/system. {\ttfamily fs\+Options} are passed to {\ttfamily fs.\+write\+File}. If {\ttfamily callback} is given, invokes it with an error, if any. If {\ttfamily callback} is not given, returns a \href{https://developer.mozilla.org/Web/JavaScript/Reference/Global_Objects/Promise}{\texttt{ {\ttfamily Promise}}} that is rejected with an error or resolved with the written virtual file.\hypertarget{readme.md_autotoc_md33020}{}\doxysubsubsection{\texorpdfstring{{\ttfamily to\+VFile.\+write\+Sync(options\mbox{[}, fs\+Options\mbox{]})}}{{\ttfamily to\+VFile.\+write\+Sync(options\mbox{[}, fs\+Options\mbox{]})}}}\label{readme.md_autotoc_md33020}
