This package is a \href{https://github.com/unifiedjs/unified}{\texttt{ unified}} (\href{https://github.com/mdx-js/recma}{\texttt{ recma}}) that enables JSX. You can use this plugin to add support for parsing and serializing it.\hypertarget{readme.md_autotoc_md27955}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md27955}
You can use this if you want to use {\ttfamily recma} and JSX. You can also use \href{https://github.com/acornjs/acorn-jsx}{\texttt{ {\ttfamily acorn-\/jsx}}} manually with \href{https://github.com/acornjs/acorn}{\texttt{ {\ttfamily acorn}}} for parsing. And use the \href{https://github.com/syntax-tree/estree-util-to-js\#jsx}{\texttt{ {\ttfamily jsx}}} handlers from \href{https://github.com/syntax-tree/estree-util-to-js}{\texttt{ {\ttfamily estree-\/util-\/to-\/js}}} manually.\hypertarget{readme.md_autotoc_md27956}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md27956}
