\hypertarget{readme.md_autotoc_md36287}{}\doxysubsubsubsection{\texorpdfstring{Why does the spinner freeze?}{Why does the spinner freeze?}}\label{readme.md_autotoc_md36287}
Java\+Script is single-\/threaded, so any synchronous operations will block the spinner\textquotesingle{}s animation. To avoid this, prefer using asynchronous operations.\hypertarget{readme.md_autotoc_md36288}{}\doxysubsubsection{\texorpdfstring{Comparison with \href{https://github.com/sindresorhus/ora}{\texttt{ {\ttfamily ora}}}}{Comparison with \href{https://github.com/sindresorhus/ora}{\texttt{ {\ttfamily ora}}}}}\label{readme.md_autotoc_md36288}
Ora offers more options, greater customizability, \href{https://github.com/sindresorhus/ora?tab=readme-ov-file\#orapromiseaction-options}{\texttt{ promise handling}}, and better Unicode detection. It’s a more mature and feature-\/rich package that handles more edge cases but comes with additional dependencies and a larger size. In contrast, this package is smaller, simpler, and optimized for minimal overhead, making it ideal for lightweight projects where dependency size is important. However, Ora is generally the better choice for most use cases.\hypertarget{readme.md_autotoc_md36289}{}\doxysubsubsection{\texorpdfstring{Related}{Related}}\label{readme.md_autotoc_md36289}
