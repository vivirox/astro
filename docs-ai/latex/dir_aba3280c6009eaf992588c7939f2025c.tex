This package is a utility that can turn a parse5 tree into a hast tree.\hypertarget{readme.md_autotoc_md16677}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md16677}
You can use this package when using {\ttfamily parse5} as an HTML parser and wanting to work with hast.

The utility \href{https://github.com/syntax-tree/hast-util-to-parse5}{\texttt{ {\ttfamily hast-\/util-\/to-\/parse5}}} does the inverse of this utility. It generates {\ttfamily parse5}s AST again.

The utility \href{https://github.com/syntax-tree/hast-util-from-html}{\texttt{ {\ttfamily hast-\/util-\/from-\/html}}} wraps this utility and {\ttfamily parse5} to both parse HTML and generate hast from it.\hypertarget{readme.md_autotoc_md16678}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md16678}
