\href{https://travis-ci.org/zertosh/loose-envify}{\texttt{ }}

Fast (and loose) selective {\ttfamily process.\+env} replacer using \href{https://github.com/lydell/js-tokens}{\texttt{ js-\/tokens}} instead of an AST. Works just like \href{https://github.com/hughsk/envify}{\texttt{ envify}} but much faster.\hypertarget{README.md_autotoc_md22074}{}\doxysubsection{\texorpdfstring{Gotchas}{Gotchas}}\label{README.md_autotoc_md22074}
