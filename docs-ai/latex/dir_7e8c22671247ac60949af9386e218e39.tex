\hypertarget{README.md_autotoc_md11613}{}\doxysubsubsubsection{\texorpdfstring{{\ttfamily escape\+Special\+Characters}}{{\ttfamily escape\+Special\+Characters}}}\label{README.md_autotoc_md11613}
Java\+Script string tags by default add an extra {\ttfamily \textbackslash{}} escape in front of some special characters such as {\ttfamily \$} dollar signs. {\ttfamily dedent} will escape those special characters when called as a string tag.

If you\textquotesingle{}d like to change the behavior, an {\ttfamily escape\+Special\+Characters} option is available. It defaults to\+:


\begin{DoxyItemize}
\item {\ttfamily false}\+: when {\ttfamily dedent} is called as a function
\item {\ttfamily true}\+: when {\ttfamily dedent} is called as a string tag
\end{DoxyItemize}
