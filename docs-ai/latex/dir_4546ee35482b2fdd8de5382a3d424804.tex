\item {\ttfamily non\+Ascii\+Printable\+Only} everything outiside of the \href{https://en.wikipedia.org/wiki/ASCII\#Printable_characters}{\texttt{ ASCII printable characters}} keeping HTML special characters intact.
\item {\ttfamily extensive} encodes all non-\/printable characters, non-\/\+ASCII characters and all characters with named references.
