{\ttfamily @vercel/analytics} allows you to track page views and custom events in your Next.\+js app or any other website that is deployed to Vercel.

All page views are automatically tracked in your app.

This package does {\bfseries{not}} track data in development mode.\hypertarget{README.md_autotoc_md7149}{}\doxysubsubsection{\texorpdfstring{Quickstart}{Quickstart}}\label{README.md_autotoc_md7149}

\begin{DoxyEnumerate}
\item Enable Vercel Web Analytics for a project in the \href{https://vercel.com/dashboard}{\texttt{ Vercel Dashboard}}.
\item Add the {\ttfamily @vercel/analytics} package to your project
\item Inject the Analytics script to your app
\begin{DoxyItemize}
\item If you are using {\bfseries{Next.\+js}} or {\bfseries{React}}, you can use the {\ttfamily \texorpdfstring{$<$}{<}Analytics /\texorpdfstring{$>$}{>}} component to inject the script into your app.
\item To add the tracking script for other frameworks, use the {\ttfamily inject} function.
\item If you want to use Vercel Web Analytics on a static site without npm, follow the instructions in the \href{https://vercel.com/docs/analytics/quickstart}{\texttt{ documentation}}.
\end{DoxyItemize}
\item Deploy your app to Vercel and see data flowing in.
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md7150}{}\doxysubsubsection{\texorpdfstring{Documentation}{Documentation}}\label{README.md_autotoc_md7150}
