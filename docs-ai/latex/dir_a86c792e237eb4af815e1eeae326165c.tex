\doxysubsection{Detailed Description}
This package is used to collect anonymous telemetry data within the Astro CLI.

It can be disabled in Astro using either method documented below\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\#\ Option\ 1:\ Run\ this\ to\ disable\ telemetry\ globally\ across\ your\ entire\ machine.}
\DoxyCodeLine{astro\ telemetry\ disable}

\end{DoxyCode}



\begin{DoxyCode}{0}
\DoxyCodeLine{\#\ Option\ 2:\ The\ ASTRO\_TELEMETRY\_DISABLED\ environment\ variable\ disables\ telemetry\ when\ set.}
\DoxyCodeLine{ASTRO\_TELEMETRY\_DISABLED=1\ astro\ dev}

\end{DoxyCode}


Visit \href{https://astro.build/telemetry/}{\texttt{ https\+://astro.\+build/telemetry/}} for more information about our approach to anonymous telemetry in Astro.
