Vendored dependencies for Victory.\hypertarget{README.md_autotoc_md34931}{}\doxysubsection{\texorpdfstring{Background}{Background}}\label{README.md_autotoc_md34931}
D3 has released most of its libraries as ESM-\/only. This means that consumers in Node.\+js applications can no longer just {\ttfamily require()} anything with a d3 transitive dependency, including much of Victory.

To help provide an easy path to folks still using Common\+JS in their Node.\+js applications that consume Victory, we now provide this package to vendor in various d3-\/related packages.\hypertarget{README.md_autotoc_md34932}{}\doxysubsection{\texorpdfstring{Packages}{Packages}}\label{README.md_autotoc_md34932}

\item d3-\/interpolate

\item d3-\/timer
