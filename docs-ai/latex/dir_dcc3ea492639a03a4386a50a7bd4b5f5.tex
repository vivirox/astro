This package is a utility that takes a \href{https://github.com/syntax-tree/hast}{\texttt{ hast}} (HTML) syntax tree as input and turns it into an \href{https://github.com/estree/estree}{\texttt{ estree}} (Java\+Script) syntax tree (with a JSX extension). Typically, there is only one node in {\ttfamily body}, however, this utility also supports embedded MDX nodes in the HTML (when \href{https://github.com/syntax-tree/mdast-util-mdx}{\texttt{ {\ttfamily mdast-\/util-\/mdx}}} is used with mdast to parse markdown before passing its nodes through to hast). When MDX ESM import/exports are used, those nodes are added before the fragment or element in body.
