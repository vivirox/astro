\doxysubsection{Detailed Description}
{\itshape Easing} is a method of distorting time to control apparent motion in animation. It is most commonly used for \href{https://en.wikipedia.org/wiki/12_basic_principles_of_animation\#Slow_In_and_Slow_Out}{\texttt{ slow-\/in, slow-\/out}}. By easing time, \href{https://github.com/d3/d3-transition}{\texttt{ animated transitions}} are smoother and exhibit more plausible motion.

The easing types in this module implement the ease method, which takes a normalized time {\itshape t} and returns the corresponding “eased” time {\itshape tʹ}. Both the normalized time and the eased time are typically in the range \mbox{[}0,1\mbox{]}, where 0 represents the start of the animation and 1 represents the end; some easing types, such as \doxylink{README.md_easeElastic}{elastic}, may return eased times slightly outside this range. A good easing type should return 0 if {\itshape t} = 0 and 1 if {\itshape t} = 1. See the \href{https://observablehq.com/@d3/easing}{\texttt{ easing explorer}} for a visual demonstration.

These easing types are largely based on work by \href{http://robertpenner.com/easing/}{\texttt{ Robert Penner}}.\hypertarget{README.md_autotoc_md11222}{}\doxysubsection{\texorpdfstring{Installing}{Installing}}\label{README.md_autotoc_md11222}

\# {\itshape ease}({\itshape t})

Given the specified normalized time {\itshape t}, typically in the range \mbox{[}0,1\mbox{]}, returns the “eased” time {\itshape tʹ}, also typically in \mbox{[}0,1\mbox{]}. 0 represents the start of the animation and 1 represents the end. A good implementation returns 0 if {\itshape t} = 0 and 1 if {\itshape t} = 1. See the \href{https://observablehq.com/@d3/easing}{\texttt{ easing explorer}} for a visual demonstration. For example, to apply \doxylink{README.md_easeCubic}{cubic} easing\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ te\ =\ d3.easeCubic(t);}

\end{DoxyCode}


Similarly, to apply custom \doxylink{README.md_easeElastic}{elastic} easing\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{//\ Before\ the\ animation\ starts,\ create\ your\ easing\ function.}
\DoxyCodeLine{const\ customElastic\ =\ d3.easeElastic.period(0.4);}
\DoxyCodeLine{}
\DoxyCodeLine{//\ During\ the\ animation,\ apply\ the\ easing\ function.}
\DoxyCodeLine{const\ te\ =\ customElastic(t);}

\end{DoxyCode}
