This package is a utility that can make a tree that potentially contains dangerous user content safe for use. It defaults to what Git\+Hub does to clean unsafe markup, but you can change that.\hypertarget{readme.md_autotoc_md17171}{}\doxysubsection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{readme.md_autotoc_md17171}
This package is needed whenever you deal with potentially dangerous user content.

The plugin \href{https://github.com/rehypejs/rehype-sanitize}{\texttt{ {\ttfamily rehype-\/sanitize}}} wraps this utility to also sanitize HTML at a higher-\/level (easier) abstraction.\hypertarget{readme.md_autotoc_md17172}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md17172}

\hypertarget{readme.md_autotoc_md17191}{}\doxysubsection{\texorpdfstring{Types}{Types}}\label{readme.md_autotoc_md17191}
This package is fully typed with \href{https://www.typescriptlang.org}{\texttt{ Type\+Script}}. It exports the additional type \`{}\+Schema\`{}.\hypertarget{readme.md_autotoc_md17192}{}\doxysubsection{\texorpdfstring{Compatibility}{Compatibility}}\label{readme.md_autotoc_md17192}
Projects maintained by the unified collective are compatible with maintained versions of Node.\+js.

When we cut a new major release, we drop support for unmaintained versions of Node. This means we try to keep the current release line, {\ttfamily hast-\/util-\/sanitize@\texorpdfstring{$^\wedge$}{\string^}5}, compatible with Node.\+js 16.\hypertarget{readme.md_autotoc_md17193}{}\doxysubsection{\texorpdfstring{Security}{Security}}\label{readme.md_autotoc_md17193}
By default, {\ttfamily hast-\/util-\/sanitize} will make everything safe to use. Assuming you understand that certain attributes (including a limited set of classes) can be generated by users, and you write your CSS (and JS) accordingly. When used incorrectly, deviating from the defaults can open you up to a \href{https://en.wikipedia.org/wiki/Cross-site_scripting}{\texttt{ cross-\/site scripting (XSS)}} attack.

Use {\ttfamily hast-\/util-\/sanitize} after the last unsafe thing\+: everything after it could be unsafe (but is fine if you do trust it).\hypertarget{readme.md_autotoc_md17194}{}\doxysubsection{\texorpdfstring{Related}{Related}}\label{readme.md_autotoc_md17194}
