\doxysubsection{Detailed Description}
Native access to Mac\+OS FSEvents in \href{https://nodejs.org/}{\texttt{ Node.\+js}}

The FSEvents API in Mac\+OS allows applications to register for notifications of changes to a given directory tree. It is a very fast and lightweight alternative to kqueue.

This is a low-\/level library. For a cross-\/platform file watching module that uses fsevents, check out \href{https://github.com/paulmillr/chokidar}{\texttt{ Chokidar}}.\hypertarget{README.md_autotoc_md15941}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{README.md_autotoc_md15941}
