Deep Eql is a module which you can use to determine if two objects are "{}deeply"{} equal -\/ that is, rather than having referential equality ({\ttfamily a === b}), this module checks an object\textquotesingle{}s keys recursively, until it finds primitives to check for referential equality. For more on equality in Java\+Script, read \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Comparison_Operators}{\texttt{ the comparison operators article on mdn}}.

As an example, take the following\+:

\DoxyCodeLine{1\ ==\ '1'\ //\ These\ are\ two\ different\ primitives,\ through\ type\ coercion\ they\ hold\ the\ same\ value\ -\/\ they\ are\ loosely\ equal}
\DoxyCodeLine{\{\ a:\ 1\ \}\ !==\ \{\ a:\ 1\ \}\ //\ These\ are\ two\ different\ objects,\ they\ hold\ different\ references\ and\ so\ are\ not\ strictly\ equal\ -\/\ even\ though\ they\ hold\ the\ same\ values\ inside}
\DoxyCodeLine{\{\ a:\ 1\ \}\ !=\ \{\ a:\ 1\ \}\ //\ They\ have\ the\ same\ type,\ meaning\ loose\ equality\ performs\ the\ same\ check\ as\ strict\ equality\ -\/\ they\ are\ still\ not\ equal.}
