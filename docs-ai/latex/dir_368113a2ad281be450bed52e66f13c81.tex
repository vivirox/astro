\doxysubsection{Detailed Description}
Click on {\ttfamily Use this template} to copy the Mintlify starter kit. The starter kit contains examples including


\begin{DoxyItemize}
\item Guide pages
\item Navigation
\item Customizations
\item API Reference pages
\item Use of popular components
\end{DoxyItemize}\hypertarget{README.md_autotoc_md37313}{}\doxysubsubsection{\texorpdfstring{Development}{Development}}\label{README.md_autotoc_md37313}
Install the \href{https://www.npmjs.com/package/mintlify}{\texttt{ Mintlify CLI}} to preview the documentation changes locally. To install, use the following command


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ i\ -\/g\ mintlify}

\end{DoxyCode}


Run the following command at the root of your documentation (where mint.\+json is)


\begin{DoxyCode}{0}
\DoxyCodeLine{mintlify\ dev}

\end{DoxyCode}
\hypertarget{README.md_autotoc_md37314}{}\doxysubsubsection{\texorpdfstring{Publishing Changes}{Publishing Changes}}\label{README.md_autotoc_md37314}
Install our Github App to auto propagate changes from your repo to your deployment. Changes will be deployed to production automatically after pushing to the default branch.
