In VS Code, there are two ways of loading custom HTML datasets\+:


\begin{DoxyEnumerate}
\item With setting {\ttfamily html.\+custom\+Data}
\item With an extension that contributes {\ttfamily contributes.\+html.\+custom\+Data}
\end{DoxyEnumerate}

Both setting point to a list of JSON files. This document describes the shape of the JSON files.

You can read more about custom data at\+: \href{https://github.com/microsoft/vscode-custom-data}{\texttt{ https\+://github.\+com/microsoft/vscode-\/custom-\/data}}.\hypertarget{md_node__modules_2vscode-html-languageservice_2docs_2custom_data_autotoc_md35303}{}\doxysection{\texorpdfstring{Custom Data Format}{Custom Data Format}}\label{md_node__modules_2vscode-html-languageservice_2docs_2custom_data_autotoc_md35303}
The JSON have one required property, {\ttfamily version} and 3 other top level properties\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ "{}version"{}:\ 1.1,}
\DoxyCodeLine{\ \ "{}tags"{}:\ [],}
\DoxyCodeLine{\ \ "{}globalAttributes"{}:\ [],}
\DoxyCodeLine{\ \ "{}valueSets"{}:\ []}
\DoxyCodeLine{\}}

\end{DoxyCode}


Version denotes the schema version you are using. The latest schema version is {\ttfamily V1.\+1}.

You can find other properties\textquotesingle{} shapes at \href{../src/htmlLanguageTypes.ts}{\texttt{ html\+Language\+Types.\+ts}} or the \href{./customData.schema.json}{\texttt{ JSON Schema}}.

When working with VSCode, you should suffix your custom data file with {\ttfamily .html-\/data.\+json}, so VS Code will load the most recent schema for the JSON file.

\href{../src/languageFacts/data/webCustomData.ts}{\texttt{ html5.\+ts}} contains that built-\/in dataset that conforms to the spec.\hypertarget{md_node__modules_2vscode-html-languageservice_2docs_2custom_data_autotoc_md35304}{}\doxysection{\texorpdfstring{Language Features}{Language Features}}\label{md_node__modules_2vscode-html-languageservice_2docs_2custom_data_autotoc_md35304}
Custom data receives the following language features\+:


\begin{DoxyItemize}
\item Completion on tag, attribute and attribute value
\item Hover on tag (here\textquotesingle{}s the \href{https://github.com/Microsoft/vscode-html-languageservice/issues/47}{\texttt{ issue}} for hover on attribute / attribute-\/name)
\end{DoxyItemize}
