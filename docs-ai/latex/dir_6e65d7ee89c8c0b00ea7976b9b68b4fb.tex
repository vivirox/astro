\doxysubsection{Detailed Description}
There are several requirements for generating the metrics used by Ka\+TeX.


\begin{DoxyItemize}
\item You need to have an installation of TeX which supports kpathsea. You can check this by running {\ttfamily tex -\/-\/version}, and seeing if it has a line that looks like \begin{quote}
kpathsea version 6.\+2.\+0 \end{quote}

\end{DoxyItemize}


\begin{DoxyItemize}
\item You need the Perl module {\ttfamily JSON}. You can install this either from CPAN (e.\+g. using the {\ttfamily cpan} command line tool\+: {\ttfamily cpan install JSON}) or with your package manager.
\item You need the Python module {\ttfamily fonttools}. You can install this either from Py\+PI (using {\ttfamily easy\+\_\+install} or {\ttfamily pip}\+: {\ttfamily pip install fonttools}) or with your package manager.
\end{DoxyItemize}

Once you have these things, run the following command from the root directory\+: \begin{DoxyVerb}sh ./dockers/fonts/buildMetrics.sh
\end{DoxyVerb}


which should generate new metrics and place them into {\ttfamily font\+Metrics\+Data.\+json}. You\textquotesingle{}re done!
