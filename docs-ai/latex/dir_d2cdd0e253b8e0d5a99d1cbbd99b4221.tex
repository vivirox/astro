The data in this repository is used in MDN Web Docs to build \href{https://developer.mozilla.org/en-US/docs/Web/CSS/background}{\texttt{ information boxes}} and \href{https://developer.mozilla.org/en-US/docs/Web/API/Window}{\texttt{ sidebar navigation}}. External tools make use of this data as well, for example, the \href{https://github.com/csstree/csstree/}{\texttt{ CSSTree}} CSS parser.

There\textquotesingle{}s a top-\/level directory for each broad area covered\+: for example, {\ttfamily api} and {\ttfamily css}. Inside each of these directories is one or more JSON files containing the data.\hypertarget{README.md_autotoc_md11152}{}\doxysubsubsection{\texorpdfstring{api}{api}}\label{README.md_autotoc_md11152}
