Basic

Constructor

Syntax

js
const color = new MooColor(str = '');
  • Parameters
    `string`str - parsable color string. If not specify this value, then color set to #000 (black).
  • Returns
    MooColor - Instance of MooColor

Parsable color string

js
'red' // named color.
'#ff0000' // Hex (full)
'#ff000080' // Hex (full with alpha)
'#f00' // Hex (shorthand)
'#f008' // Hex (shorthand with alpha)
'rgb(255, 0, 0)' // rgb
'rgba(255, 0, 0, 0.5)' // rgba
'rgb(100%, 0%, 0%)' // rgb (percent)
'rgba(100%, 0%, 0%, 0.5)' // rgba (percent)
'hsl(0, 100%, 50%)' // hsl
'hsla(0, 100%, 50%, 0.5)' // hsla
'hwb(0, 0%, 0%)' // hwb
'hwb(0, 0%, 0%, 0.5)' // hwb with alpha
'hsv(0, 100%, 100%)' // hsv
'hsv(0, 100%, 100%, 0.5)' // hsv with alpha
'cmyk(0%, 100%, 100%, 0%)' // cmyk
'cmyk(0%, 100%, 100%, 0%, 0.5)' // cmyk with alpha

Static

  • mix: Helper method for mix() method.
  • random: Create random color as HWB model.

Setter

  • setColor: Set color data.
  • getColor: Get color data.
  • getColorAs: Get color data as specific color model.
  • getModel: Get color model name.
  • changeModel: Converts color data to given color model.
  • getAlpha: Get alpha value from Color.
  • setAlpha: Set alpha value to Color.

Formatter

  • toString: Represents color as notation of specific color model.
  • toHex: Represents color as HEX notation.
  • toRgb: Represents color as RGB notation.
  • toHwb: Represents color as HWB notation.
  • toHsl: Represents color as HSL notation.
  • toHsv: Represents color as HSV notation. This format is similar to HSL.
  • toCmyk: Represents color as CMYK notation.

State access

  • brightness: readonly Returns color brightness from 0 to 255.
  • isLight: readonly Returns whether color is light or not.
  • isDark: readonly Returns whether color is dark or not.
  • luminance: readonly Returns luminance value of the color. range from 0 to 1.
  • contrastRatioWith: Returns contrast ratio with other color. range from 0 to 21.
  • isContrastEnough: Return true if contrast ratio >= 4.5

Modification

  • lighten: Increase lightness.
  • darken: Decrease darkness.
  • saturate: Increase saturation.
  • desaturate: Decrease saturation.
  • grayscale: Sets saturation value to 0.
  • whiten: Modify whiteness.
  • blacken: Modify blackness.
  • rotate: Rotate hue value.
  • mix: Mix two colors.
  • complement: Sets color to the complement of a color.
  • invert: Sets color to the inverse (negative) of a color.
